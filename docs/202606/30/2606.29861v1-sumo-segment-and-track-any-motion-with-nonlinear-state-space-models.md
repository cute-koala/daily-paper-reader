## 摘要
视觉目标跟踪（VOT）和运动对象分割（MOS）是计算机视觉中两个涉及时空对象动态的基本任务。现有方法主要依赖视觉线索，因此在现实场景中，当对象运动本身具有复杂且非线性的特性时，往往表现不佳。为解决这一局限，我们提出SUMO——一个零样本、无需训练的统一框架，将非线性动力学与基于视觉的分割相结合，实现准确且一致的VOT和MOS。具体而言，我们受机器人学原理启发，开发了一个非线性状态空间模型（SSM）来捕捉复杂的对象动态。在此基础上，我们提出了一种选择性无迹卡尔曼滤波器（SUF）用于精确状态估计，该滤波器具有联合评分机制，并动态融合多源预测以识别随时间变化的最可能的对象状态。此外，我们应用记忆选择机制来评估记忆帧的可靠性。广泛的实验结果表明，SUMO在VOT和MOS任务上均达到了最先进的性能。

## Abstract
Visual Object Tracking (VOT) and Moving Object Segmentation (MOS) are two fundamental tasks in computer vision that involve both spatial and temporal object dynamics. Existing methods rely predominantly on visual cues and thus often falter in real-world scenarios where object motions are inherently complex and nonlinear. To address this limitation, we propose SUMO, a zero-shot, training-free, unified framework integrating nonlinear dynamics with vision-based segmentation for accurate and consistent VOT and MOS. Specifically, we develop a nonlinear State Space Model (SSM) inspired by robotics principles to capture the complex object dynamics. Building on this model, we propose a Selective Unscented Filter (SUF) for accurate state estimation, which features a joint scoring mechanism and dynamically fuses multi-source predictions to identify the most plausible object state over time. Furthermore, we apply a memory selection mechanism to evaluate the reliability of memory frames. Our extensive experimental results show that SUMO achieves state-of-the-art performance on both VOT and MOS tasks.

#### 2. 论文提出的方法论
- **核心思想**：提出SUMO，一个**零样本、无需训练**的统一框架。通过将**非线性状态空间模型（SSM）** 与基于视觉的分割模型（SAM2）以即插即用的方式相结合，来同时实现准确的VOT和MOS。
- **关键技术细节**：
    1.  **非线性状态空间模型 (Nonlinear SSM)**：
        - 状态定义：定义了一个8维状态向量 **x** = [x, y, a, h, v, θ, v_a, v_h]^T，其中 (x, y) 为边界框中心， (a, h) 为长宽比和高度， v 和 θ 为速度和运动方向， v_a 和 v_h 为形状变化率。这同时建模了物体的运动轨迹和形状演变。
        - 非线性状态方程：基于机器人学原理，使用三角函数（cosθ, sinθ）来描述方向变化，使模型能够自然地建模弯曲轨迹。
    2.  **选择性无迹卡尔曼滤波器 (Selective Unscented Filter, SUF)**：
        - 问题：无法直接获取传感器测量值。
        - 解决方案：SUF在无迹变换基础上，创新地提出了一个**联合评分机制**。
            - 它将来自**非线性SSM的状态预测**（得到预测框）与**图像解码器**生成的三个候选掩膜（及对应的IoU分数）进行融合。
            - 通过计算候选框与SSM预测框的IoU（NSSM IoU），并结合解码器IoU，得到一个综合得分（Score = λ·IoU_nssm + (1-λ)·IoU_md）。当解码器置信度极低时（如遮挡），则完全依赖SSM的预测。
            - 使用一个**稳定性阈值**决定是否执行SUF的更新步骤，避免低质量观察结果污染状态估计。
        - 流程：Sigma点生成 → 非线性预测 → 联合评分选择最佳掩膜 → 状态更新。
    3.  **记忆选择机制 (Memory Selection Mechanism)**：在将过往帧的特征用于当前帧的注意力计算时，该机制会评估记忆帧的可靠性。只有满足“解码器IoU、物体置信度、NSSM IoU”均高于各自阈值的帧，才会被选入记忆库，从而避免低质量记忆对当前预测的干扰。
- **算法流程（文字描述）**：
    1.  **初始化**：根据第一帧的prompt（如边界框），生成初始掩膜，并初始化SUF的状态。
    2.  **循环处理每一帧**：
        a. **视觉编码**：图像编码器提取当前帧的多尺度特征。
        b. **记忆注意**：通过记忆选择机制，从记忆库中挑选可靠帧，与当前帧特征进行交叉注意力计算。
        c. **掩膜解码**：解码器生成三个候选掩膜及其IoU和物体置信度。
        d. **SUF预测**：利用非线性SSM，从上一帧的后验状态预测当前帧的先验状态，并映射到观察空间（得到预测框）。
        e. **联合评分与掩膜选择**：综合SSM和视觉解码器的得分，选出最佳掩膜作为当前帧输出。
        f. **SUF更新**：根据最佳掩膜的质量决定是否进行卡尔曼更新，从而得到当前帧的后验状态估计，用于下一帧预测。
        g. **构建记忆**：将当前帧的高质量特征、位置编码、物体指针存入记忆库。
