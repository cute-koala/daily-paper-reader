## 摘要
无监督视频对象跟踪旨在无需人工标注的情况下将动态场景分解为持久的、以对象为中心的实体。近年来许多方法依赖于基于槽的表示，其中一组固定的潜在变量（“槽”）代表跨帧的各个对象。为了保持对象身份，这些模型对槽嵌入施加时间一致性。然而，当外观和姿态纠缠在一起时，这种一致性目标与对象运动和视角变化相冲突。结果，槽倾向于锁定静态区域（例如背景）以满足一致性目标，而前景对象则分散到多个槽中或频繁交换身份。为了解决这些限制，我们提出了STAITUS，一个统一框架，将每个槽显式解耦为外观和几何姿态（位置/尺度）。利用这种解耦，STAITUS强制执行帧内空间分离，并仅在 appearance 空间应用时间对齐，从而在运动、遮挡和对象进入/退出下产生更清晰的掩码和更持久的身份。此外，为了减轻过分割，我们引入了一种自适应门控机制，动态调整活动槽的数量以匹配场景复杂度。在合成和真实世界基准上的大量实验表明，STAITUS在分割质量和跟踪稳定性上大幅优于最先进的基线方法。

## Abstract
Unsupervised video object tracking aims to decompose dynamic scenes into persistent, object-centric entities without manual annotations. Many recent approaches rely on slot-based representations, where a fixed set of latent variables ("slots") represent individual objects across frames. To preserve object identity, these models enforce temporal consistency on slot embeddings. However, when appearance and pose are entangled, this consistency objective conflicts with object motion and viewpoint changes. As a result, slots tend to lock onto static regions (e.g., background) to satisfy the consistency objective, while foreground objects become fragmented across multiple slots or frequently swap identities. To address these limitations, we propose STAITUS, a unified framework that explicitly disentangles each slot into appearance and geometric pose (position/scale). Leveraging this disentanglement, STAITUS enforces within-frame spatial separation and applies temporal alignment only in appearance space, yielding sharper masks and more persistent identities under motion, occlusion, and object entry/exit. Furthermore, to mitigate over-segmentation, we introduce an adaptive gating mechanism that dynamically adjusts the number of active slots to match scene complexity. Extensive experiments on synthetic and real-world benchmarks demonstrate that STAITUS substantially outperforms state-of-the-art baselines in segmentation quality and tracking stability.

## 1. 核心问题与整体含义（研究动机和背景）
- **研究动机**：无监督视频对象跟踪旨在无需人工标注的条件下，将动态场景分解为持久的、以对象为中心的实体。现有方法普遍采用“槽”（slot）表示，每个槽对应一个对象或背景，并通过强制槽嵌入的时间一致性来保持对象身份。
- **核心问题**：当槽表示同时编码了对象的**外观**和**姿态（位置/尺度）**时，强制时间一致性会与对象自身的运动、视角变化产生冲突。这导致槽倾向于“锚定”在静态背景区域以迎合一致性目标，而前景对象则被碎片化（一个对象分散到多个槽）或频繁发生身份交换（slot switching）。此外，固定数量的槽还会引发过分割（over-segmentation）。

## 2. 方法论：核心思想、关键技术细节
- **核心思想**：将每个槽显式解耦为**外观嵌入**和**几何姿态（2D位置与尺度）**，使得时间对齐仅作用于外观空间，而几何属性可自由演化以适配运动、遮挡和物体进出场景。同时引入帧内空间分离损失和自适应门控机制，提升分割精度与身份稳定性。
- **关键技术细节**：
  - **幻觉编码器**：使用预训练并冻结的 DINO ViT-B/16 提取密集特征，再通过一个轻量 MLP 投影到任务特定空间。
  - **递归自适应解耦槽注意力模块**：
    - 基于 Invariant Slot Attention (ISA) 设计，每个槽包含 (v_k, p_k, s_k)。
    - 递归初始化：利用前一帧的槽状态通过残差 MLP 预测下一帧的外观（高斯分布），位置和尺度则直接传播。
    - 自适应槽激活：每个槽的外观嵌入通过门控 MLP 预测二进制激活变量（Gumbel-Softmax 放松），仅活跃槽参与重建。
  - **自适应空间广播解码**：采用空间广播解码器，每个槽独立解码出 RGB 重建和 alpha 掩码，最后通过活跃槽加权合成最终帧。解码器利用解耦的姿态参数构建槽特定的坐标网格。
  - **时间对齐损失 L_time**：仅在活跃槽的外观嵌入上计算余弦相似度，鼓励跨帧一致性，同时允许姿态自由变化。
  - **空间分离损失 L_sep**：在同一帧内，对活跃槽的外观嵌入对惩罚正余弦相似度，防止多个槽融合到同一对象。
  - **总损失**：L = L_recon + λ_time L_time + λ_sep L_sep + λ_spars L_spars（L_spars 是稀疏正则化）。
- **算法流程**：每帧输入 → DINO编码 → MLP投影 → 递归初始化（基于前一帧） → 解耦槽注意力迭代（ISA） → 门控激活 → 独立解码 → 复合重建 → 计算损失。

## 6. 主要结论与发现
- STAITUS 在所有数据集上 consistently 优于基线，在对象发现和跟踪任务中均取得最高 ARI、FG-ARI 和 mBO。
- 解耦设计解决了传统槽表示中“时间一致性 vs 运动”的根本冲突：允许姿态自由变化，外观对齐得以有效实施，从而获得更清晰的掩码和更稳定的身份。
- 自适应门控有效减少了过分割，尤其在复杂场景中（如 YouTube-VIS 上移除 L_spars 后 ARI 从 0.23 降至 0.14）。

- 特征编码器选择影响有限，但更强编码器（DINOv3）可带来额外提升。
