## 摘要
从自然经验中学习接地词汇意义需要解决婴儿视角记录中的两个歧义：命名指称对象何时出现以及它在杂乱画面中的位置。在SAYCam风格的数据中，看护者的语言稀疏且与自我中心视频弱同步，因此单帧对比配对会产生噪声正样本，其中目标对象缺失或被干扰物纠缠。我们提出BabyMind，一种在稀疏、嘈杂监督下用于儿童视角对比学习的物体优先偏差。BabyMind使用基于离线掩码的区域接口提取候选对象嵌入，通过跟踪将候选对象在短话语中心窗口内链接成轻量级对象文件，并使用原型空间多实例对比目标将话语与对象文件包对齐。轨迹一致性和全局对象一致性正则化项稳定学习，并将对象文件结构转移到评估时使用的全局帧嵌入中。在SAYCam-S上，BabyMind将Labeled-S 15强制选择准确率比CVCL提高+2.6个百分点，并在词汇外分布外基准上取得一致增益。代码可在https://github.com/sathiiii/BabyMind获取。

## Abstract
Learning grounded word meaning from natural experience requires resolving two ambiguities in infant-view recordings: when the named referent appears and where it is in a cluttered frame. In SAYCam-style data, caregiver speech is sparse and weakly synchronized with egocentric video, so single-frame contrastive pairing yields noisy positives in which the intended object is absent or entangled with distractors. We propose BabyMind, an object-first bias for child-view contrastive learning under sparse, noisy supervision. BabyMind extracts candidate object embeddings using an offline mask-based region interface, links candidates across a short utterance-centered window into lightweight object files via tracking, and aligns utterances to bags of object files with a prototype-space multiple-instance contrastive objective. Track-coherence and global-object agreement regularizers stabilize learning and transfer object-file structure into the global frame embedding used at evaluation. On SAYCam-S, BabyMind improves Labeled-S 15 forced-choice accuracy by +2.6 points over CVCL and yields consistent gains on in-vocabulary out-of-distribution benchmarks. Code is available at https://github.com/sathiiii/BabyMind.

## 2. 论文提出的方法论

### 核心思想
引入**物体优先（object-first）**归纳偏置，在CVCL全局对比目标之外，增加一个**辅助物体文件路径**：从话语对应的短时间窗口（M帧）中提取并跟踪候选物体区域，形成物体文件包，然后将话语与物体文件包通过多实例学习（MIL）对齐，再通过正则化项将物体文件结构传递到全局帧嵌入。

### 关键技术细节

1. **离线自动掩码生成（AMG）**：使用Segment Anything模型预计算每帧的实例掩码（最多24个），下采样到特征图分辨率。若掩码在过滤后消失，则退回到特征图上的patch候选（top-K patch位置）。

2. **候选物体嵌入**：对每个掩码区域做掩膜平均池化，再通过投影头Π映射到共享嵌入空间（L2归一化）。每个帧获得一组候选嵌入。

3. **短窗口物体文件**：在话语对应的M帧窗口（锚帧+额外M-1帧）内，通过贪婪相似度跟踪（余弦相似度阈值0.55）将跨帧的候选嵌入链接成物体文件（track）。每个物体文件嵌入为分配给它的候选嵌入均值（L2归一化）。最多保留R_max=16个物体文件，外加一个可学习NULL嵌入表示“窗口中无接地指代”。

4. **原型空间多实例对比学习**：
   - 维护一个可学习原型码本P（K=64，在线EMA更新，Sinkhorn平衡）。
   - 话语嵌入和每个物体文件嵌入都转换为软原型分配向量q。
   - 定义实例相似度 a = (话语原型分配)^T (物体文件原型分配)。
   - 使用masked log-sum-exp池化聚合一个包内所有物体文件的相似度，得到包级相似度S_mil。
   - 施加对称的batch内对比损失L_mil。

5. **正则化项**：
   - **轨迹一致性损失L_coh**：物体文件内部各帧候选的原型分配应接近该物体文件整体原型分配（KL散度，stop-gradient）。
   - **全局-物体一致性损失L_go**：选择与话语最匹配的非空物体文件，用其原型混合重建嵌入，最小化该重建与全局帧嵌入的余弦距离，从而将物体文件结构转移到全局嵌入。

6. **总体目标**：L = λ_glob L_glob + λ_MIL L_mil + λ_coh L_coh + λ_go L_go，其中L_glob是CVCL的全局对比损失（保持原接口）。
