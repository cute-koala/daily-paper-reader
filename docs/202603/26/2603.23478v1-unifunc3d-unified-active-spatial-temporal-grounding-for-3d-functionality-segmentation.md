## 摘要
三维场景中的功能性分割要求智能体将隐式的自然语言指令定位到细粒度交互元素的精确掩码上。现有方法依赖于碎片化的流水线，在初始任务解析阶段存在视觉盲区。我们观察到，这些方法受限于单尺度、被动且启发式的帧选择。我们提出了 UniFunc3D，这是一个统一且无需训练的框架，它将多模态大语言模型视为一个主动观察者。通过将语义、时间及空间推理整合到单次前向传播中，UniFunc3D 进行联合推理，从而将任务分解建立在直接的视觉证据之上。我们的方法引入了采用由粗到精策略的主动时空定位。这使得模型能够自适应地选择正确的视频帧，并专注于高细节的交互部分，同时保留消除歧义所需的全局上下文。在 SceneFun3D 数据集上，UniFunc3D 达到了最先进的性能，在没有任何特定任务训练的情况下，以 59.9% 的 mIoU 相对提升大幅超越了无需训练和基于训练的方法。代码将在我们的项目页面发布：https://jiaying.link/unifunc3d。

## Abstract
Functionality segmentation in 3D scenes requires an agent to ground implicit natural-language instructions into precise masks of fine-grained interactive elements. Existing methods rely on fragmented pipelines that suffer from visual blindness during initial task parsing. We observe that these methods are limited by single-scale, passive and heuristic frame selection. We present UniFunc3D, a unified and training-free framework that treats the multimodal large language model as an active observer. By consolidating semantic, temporal, and spatial reasoning into a single forward pass, UniFunc3D performs joint reasoning to ground task decomposition in direct visual evidence. Our approach introduces active spatial-temporal grounding with a coarse-to-fine strategy. This allows the model to select correct video frames adaptively and focus on high-detail interactive parts while preserving the global context necessary for disambiguation. On SceneFun3D, UniFunc3D achieves state-of-the-art performance, surpassing both training-free and training-based methods by a large margin with a relative 59.9\% mIoU improvement, without any task-specific training. Code will be released on our project page: https://jiaying.link/unifunc3d.

### 2. 方法论
UniFunc3D 是一个**统一且无需训练（Training-free）**的框架，核心流程如下：
*   **主动时空定位（Active Spatial-Temporal Grounding）**：
    *   **粗略阶段（Round 1）**：在低分辨率下对全视频进行多轮采样（Multi-sampling），让 MLLM 识别功能物体名称并选出最具代表性的候选关键帧。
    *   **精细阶段（Round 2）**：在候选帧周围提取密集的时域窗口，并以原生高分辨率进行处理。这种“由粗到精”的策略模拟了人类视觉的缩放能力，同时保留了全局上下文以消除空间歧义。
*   **视觉掩码生成与验证**：
    *   利用 MLLM 预测的坐标点作为提示（Prompt），调用 **SAM3** 生成候选 2D 掩码。
    *   **视觉验证**：将生成的掩码以彩色覆盖图的形式反馈给 MLLM，由其直接判断该区域是否精确对应目标部件，从而过滤过分割或错误的预测。
*   **3D 提升与共识**：通过相机位姿将验证后的 2D 掩码投影回 3D 点云，并利用多视角一致性（Multi-view Agreement）机制剔除离群点，生成最终的 3D 功能掩码。

*   **性能突破**：UniFunc3D-30B 在所有指标上均达到 SOTA。在 split0 上，其 mIoU 相比 Fun3DU 提升了 **59.9%**（相对提升），AP50 提升了 14.34 个百分点。

### 7. 优点
*   **统一性**：将任务解析、时空定位和视觉验证整合在单一 MLLM 框架内，减少了碎片化流程带来的误差累积。
*   **免训练且高效**：无需昂贵的 3D 标注和训练过程，且推理速度显著快于前代方法。
*   **拟人化感知**：由粗到精的缩放机制有效解决了 3D 场景中微小交互部件难以定位的问题。

### 8. 不足与局限
*   **极端尺寸挑战**：对于占图像面积小于 0.1% 的极微小部件，定位依然存在困难。
*   **遮挡问题**：在存在严重遮挡的复杂场景中，仅靠视频序列的采样可能仍无法获得完美的视角。
*   **交互性缺失**：目前仍是单向推理过程，未来可引入用户反馈的交互式修正机制。
*   **应用限制**：虽然推理速度有所提升，但 26 分钟/场景的耗时对于实时机器人交互仍有距离。
