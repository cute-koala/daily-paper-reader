## 摘要
大型视觉语言模型 (LVLMs) 在视觉理解和推理方面表现出色，但过多的视觉 Token 导致了高昂的推理成本。尽管最近的 Token 削减方法缓解了这一问题，但它们主要针对单轮视觉问答 (VQA)，使得更具实际意义的多轮视觉问答 (MT-VQA) 场景在很大程度上尚未得到探索。MT-VQA 引入了额外的挑战，因为后续问题预先未知，且可能涉及图像的任意区域，这使得现有的削减策略失效。具体而言，当前的方法分为两类：提示依赖型 (prompt-dependent) 方法，它们偏向于初始文本提示，并丢弃了对后续轮次有用的信息；提示无关型 (prompt-agnostic) 方法，虽然在技术上适用于多轮设置，但依赖于注意力分数等启发式削减指标，导致性能欠佳。在本文中，我们提出了一种基于学习的提示无关方法，称为 MetaCompress，克服了启发式设计的局限性。我们首先将 Token 削减公式化为一种可学习的压缩映射，将剪枝 (pruning) 和合并 (merging) 等现有形式统一到一个单一的学习目标中。基于这一公式，我们引入了一种数据高效的训练范式，能够在有限的计算成本下学习最优压缩映射。在 MT-VQA 基准测试和多种 LVLM 架构上的广泛实验表明，MetaCompress 在保持跨对话轮次的强大泛化能力的同时，实现了卓越的效率-准确度权衡。我们的代码可在 https://github.com/MArSha1147/MetaCompress 获取。

## Abstract
Large Vision-Language Models (LVLMs) excel in visual understanding and reasoning, but the excessive visual tokens lead to high inference costs. Although recent token reduction methods mitigate this issue, they mainly target single-turn Visual Question Answering (VQA), leaving the more practical multi-turn VQA (MT-VQA) scenario largely unexplored. MT-VQA introduces additional challenges, as subsequent questions are unknown beforehand and may refer to arbitrary image regions, making existing reduction strategies ineffective. Specifically, current approaches fall into two categories: prompt-dependent methods, which bias toward the initial text prompt and discard information useful for subsequent turns; prompt-agnostic ones, which, though technically applicable to multi-turn settings, rely on heuristic reduction metrics such as attention scores, leading to suboptimal performance. In this paper, we propose a learning-based prompt-agnostic method, termed MetaCompress, overcoming the limitations of heuristic designs. We begin by formulating token reduction as a learnable compression mapping, unifying existing formats such as pruning and merging into a single learning objective. Upon this formulation, we introduce a data-efficient training paradigm capable of learning optimal compression mappings with limited computational costs. Extensive experiments on MT-VQA benchmarks and across multiple LVLM architectures demonstrate that MetaCompress achieves superior efficiency-accuracy trade-offs while maintaining strong generalization across dialogue turns. Our code is available at https://github.com/MArSha1147/MetaCompress.

#### 1. 核心问题与整体含义
大型视觉语言模型 (LVLM) 在处理高分辨率图像或多尺度输入时，会产生数以千计的视觉 Token，导致推理时的计算开销（尤其是注意力机制的 $O(n^2)$ 复杂度）和内存占用巨大。
*   **研究动机**：现有的 Token 削减方法主要针对单轮 VQA。在更具实际意义的**多轮对话 (MT-VQA)** 场景下，后续问题不可预测，现有的“提示依赖型”（易丢失后续轮次信息）和“提示无关型”（依赖启发式注意力分数，性能非最优）方法均表现不佳。
*   **核心目标**：提出一种基于学习的、提示无关的 Token 削减方法，以在多轮对话中实现更高的效率与准确率平衡。

#### 2. 方法论
论文提出了 **MetaCompress**，其核心思想是将 Token 削减建模为一个可学习的压缩映射过程：
*   **统一形式化**：将剪枝（Pruning）和合并（Merging）统一为线性投影 $\tilde{X}_{IMG} = P X_{IMG}$，其中 $P$ 是稀疏压缩矩阵。
*   **Meta 生成器 ($P_{meta}$)**：设计了一个轻量级模块，仅根据图像序列 $X_{IMG}$ 生成压缩矩阵 $P$。
    *   **关键技术**：通过对图像序列进行空间下采样生成查询（Query），并对原始序列进行线性投影生成键（Key），计算两者的加权内积并经过 Softmax 归一化得到 $P$。
    *   **动态分辨率适配**：利用分数阶最大池化（Fractional Max Pooling）处理不同分辨率产生的变长 Token 序列。
*   **训练范式**：采用数据高效的训练方式，优化目标包含三部分：
    1.  **预测差异损失 ($L_{pred}$)**：最小化压缩前后模型输出分布的 KL 散度。
    2.  **稀疏正则化 ($L_{entropy}$)**：鼓励压缩矩阵的稀疏性。
    3.  **崩溃正则化 ($L_{collapse}$)**：防止所有压缩后的 Token 仅来源于少数原始 Token。

#### 6. 主要结论与发现
*   **启发式指标的局限性**：通过实验发现，被保留的最优 Token 与传统的启发式指标（如 [CLS] Token 注意力或提示词注意力）并无显著相关性，证明了学习型方法的必要性。
*   **性能优越性**：MetaCompress 在 90% 的高削减率下，准确率显著优于 FastV 和 PruMerge，且在多轮对话的后续轮次中表现出更强的稳定性。
*   **效率提升**：在保持较高准确率的同时，计算开销（TFLOPs）可降低至原来的 1/5 左右，显存占用也有明显下降。

#### 7. 优点
*   **提示无关性**：非常适合多轮对话场景，一次压缩即可应对后续所有未知问题。
*   **轻量且高效**：Meta 生成器参数极少，训练成本低，且推理时引入的额外延迟几乎可以忽略。
*   **通用性强**：能够无缝集成到采用多尺度视觉特征的现代 LVLM（如 LLaVA-NeXT）中，且具备良好的跨任务迁移能力。

#### 8. 不足与局限
*   **训练依赖**：虽然是轻量化训练，但相比于 FastV 等“即插即用”的无监督方法，仍需要少量的训练数据和计算资源。
*   **插入位置限制**：目前主要实验集中在 LLM 解码器之前进行一次性压缩，尚未深入探讨在 LLM 内部多层进行逐层压缩的潜力。
*   **极端削减下的性能损失**：在 95% 以上的极端削减率下，虽然优于基准，但绝对准确率仍有较明显的下降，对于极度精细的视觉推理任务可能存在风险。
