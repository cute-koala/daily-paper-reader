## 摘要
大规模视觉语言模型（LVLMs）在需要多模态交互的一系列下游任务中取得了显著成功，但其强大的能力也带来了巨大的计算和内存开销，阻碍了实际部署。在众多加速技术中，训练后量化（PTQ）是降低内存成本和加速推理的一种流行且有效的策略。然而，现有的 LVLM 量化方法通常在模态级别衡量 token 敏感度，这无法捕捉复杂的跨 token 交互，也无法在 token 级别定量衡量量化误差。随着 token 在模型内部进行交互，模态之间的区别逐渐减弱，这表明需要进行细粒度校准。受机械可解释性中公理化归因的启发，我们提出了一种基于量化感知积分梯度（QIG）的细粒度量化策略。该策略利用积分梯度定量评估 token 敏感度，并将粒度从模态级别推进到 token 级别，从而反映模态间和模态内的动态变化。在 W4A8 和 W3A16 设置下对多个 LVLM 进行的广泛实验表明，我们的方法在不同模型和基准测试中均提高了准确率，且延迟开销可以忽略不计。例如，在 3-bit 仅权重量化下，我们的方法将 LLaVA-onevision-7B 的平均准确率提高了 1.60%，将其与全精度模型的差距缩小至仅 1.33%。代码可在 https://github.com/ucas-xiang/QIG 获取。

## Abstract
Large Vision Language Models (LVLMs) have achieved remarkable success in a range of downstream tasks that require multimodal interaction, but their capabilities come with substantial computational and memory overhead, which hinders practical deployment. Among numerous acceleration techniques, post-training quantization is a popular and effective strategy for reducing memory cost and accelerating inference. However, existing LVLM quantization methods typically measure token sensitivity at the modality level, which fails to capture the complex cross-token interactions and falls short in quantitatively measuring the quantization error at the token level. As tokens interact within the model, the distinction between modalities gradually diminishes, suggesting the need for fine-grained calibration. Inspired by axiomatic attribution in mechanistic interpretability, we introduce a fine-grained quantization strategy on Quantization-aware Integrated Gradients (QIG), which leverages integrated gradients to quantitatively evaluate token sensitivity and push the granularity from modality level to token level, reflecting both inter-modality and intra-modality dynamics. Extensive experiments on multiple LVLMs under both W4A8 and W3A16 settings show that our method improves accuracy across models and benchmarks with negligible latency overhead. For example, under 3-bit weight-only quantization, our method improves the average accuracy of LLaVA-onevision-7B by 1.60%, reducing the gap to its full-precision counterpart to only 1.33%. The code is available at https://github.com/ucas-xiang/QIG.

#### 1. 核心问题与整体含义（研究动机和背景）
大型视觉语言模型（LVLM）虽然在多模态任务中表现卓越，但其巨大的参数量和计算开销限制了在边缘设备上的实际部署。**训练后量化（PTQ）**是降低内存和加速推理的关键技术。
*   **研究动机**：现有的 LVLM 量化方法（如 MBQ）主要在“模态级别”（如区分视觉 Token 和文本 Token）衡量量化敏感度。
*   **核心发现**：作者通过实验发现，Token 的敏感度不仅在模态间存在差异，在**模态内部（Intra-modality）**以及**不同网络深度**之间也表现出极大的异质性。简单的模态级加权无法捕捉复杂的 Token 间交互，导致量化后的精度损失较大。

#### 2. 方法论：核心思想与技术细节
论文提出了一种名为 **QIG (Quantization-aware Integrated Gradients)** 的细粒度量化策略，旨在将敏感度评估从模态级推进到 Token 级。
*   **核心思想**：借鉴机械可解释性中的“公理化归因”理论，利用积分梯度（Integrated Gradients）来定量衡量每个输入 Token 对“量化引起的输出扰动”的贡献度。
*   **关键技术细节**：
    *   **QIG 定义**：不同于传统的积分梯度（解释模型预测），QIG 解释的是**全精度模型与量化模型之间的输出差异**。它计算从量化参考输入到原始输入路径上的梯度积分。
    *   **IQR 裁剪（Robust Clipping）**：由于原始 QIG 值通常呈现重尾分布（存在极端离群值），作者引入了基于四分位距（IQR）的裁剪方法来稳定敏感度估计，防止少数 Token 主导优化过程。
    *   **细粒度校准目标**：将计算出的 Token 权重 $\lambda_i$ 引入通道等化（Channel-wise Equalization, CWE）的优化目标中，使模型在校准过程中优先保护那些对量化误差贡献最大的敏感 Token。

#### 7. 优点（亮点）
*   **理论扎实**：将可解释性研究中的归因理论（积分梯度）巧妙地应用于量化领域，提供了坚实的数学支撑。
*   **粒度精细**：突破了以往“视觉 vs 文本”的粗放分类，实现了对特殊 Token、关键视觉区域和核心文本词汇的差异化保护。
*   **高效实用**：在几乎不增加推理延迟和极小校准开销的前提下，显著提升了低比特量化模型的性能。

*   **校准集依赖**：虽然仅需 128 个样本，但量化效果仍可能受到校准数据分布的影响（如 OCR 任务需要特定的校准数据）。

*   **应用范围**：目前主要验证了 Transformer 架构的 LVLM，对于其他新兴架构（如基于 Mamba 的多模态模型）尚未进行验证。
