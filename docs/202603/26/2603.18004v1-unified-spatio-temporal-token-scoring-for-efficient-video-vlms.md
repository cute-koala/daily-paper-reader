## 摘要
Token 剪枝对于提高视觉语言模型 (VLM) 的计算效率至关重要，特别是在时间冗余普遍存在的视频任务中。先前的方法通常要么 (1) 仅在视觉 Transformer (ViT) 内部针对动作识别和物体分割等单模态感知任务进行 Token 剪枝，而不适应下游的视觉语言任务；要么 (2) 仅在 LLM 内部进行剪枝，同时保持 ViT 输出完整，这通常需要复杂的文本条件 Token 选择机制。在本文中，我们提出了时空 Token 评分 (STTS)，这是一个简单且轻量级的模块，它在 ViT 和 LLM 中跨架构剪枝视觉 Token，无需文本条件或 Token 合并，并且完全兼容端到端训练。通过辅助损失学习时间评分，并通过 LLM 下游梯度学习空间评分，在高效打包算法的辅助下，STTS 在整个架构中剪枝了 50% 的视觉 Token，在 13 个长短视频问答 (QA) 任务中平均性能仅下降 0.7% 的情况下，使训练和推理效率均提升了 62%。随着每个视频采样帧数的增加，效率提升更加显著。在长视频问答中应用测试时缩放 (test-time scaling) 相比基线进一步获得了 0.5-1% 的性能提升。总的来说，STTS 代表了一种新颖、简单且有效的技术，用于统一的、全架构范围的视觉 Token 剪枝。

## Abstract
Token pruning is essential for enhancing the computational efficiency of vision-language models (VLMs), particularly for video-based tasks where temporal redundancy is prevalent. Prior approaches typically prune tokens either (1) within the vision transformer (ViT) exclusively for unimodal perception tasks such as action recognition and object segmentation, without adapting to downstream vision-language tasks; or (2) only within the LLM while leaving the ViT output intact, often requiring complex text-conditioned token selection mechanisms. In this paper, we introduce Spatio-Temporal Token Scoring (STTS), a simple and lightweight module that prunes vision tokens across both the ViT and the LLM without text conditioning or token merging, and is fully compatible with end-to-end training. By learning how to score temporally via an auxiliary loss and spatially via LLM downstream gradients, aided by our efficient packing algorithm, STTS prunes 50% of vision tokens throughout the entire architecture, resulting in a 62% improvement in efficiency during both training and inference with only a 0.7% drop in average performance across 13 short and long video QA tasks. Efficiency gains increase with more sampled frames per video. Applying test-time scaling for long-video QA further yields performance gains of 0.5-1% compared to the baseline. Overall, STTS represents a novel, simple yet effective technique for unified, architecture-wide vision token pruning.

### 论文总结：Unified Spatio-Temporal Token Scoring for Efficient Video VLMs (STTS)

#### 1. 核心问题与整体含义
视频视觉语言模型（Video VLMs）在处理长视频时面临巨大的计算开销，主要源于视频帧在空间（每帧的 Patch）和时间（帧数）维度产生的海量 Token。现有的剪枝方法要么仅针对 ViT（视觉编码器）进行单模态任务优化，要么仅在 LLM（大语言模型）端进行压缩而保留了 ViT 的计算瓶颈。
**本文核心动机**是提出一种**统一且轻量级**的剪枝方案 **STTS**，旨在不依赖复杂文本引导或 Token 合并的前提下，同时减少 ViT 和 LLM 的计算负担，实现端到端的高效视频理解。

#### 2. 方法论
STTS 模块被插入在 ViT 的中间层（默认第 3 层），通过以下三个关键技术实现：
*   **双轴评分机制（Dual-Axis Scoring）：**
    *   **空间维度：** 通过将评分作为“注意力偏置（Attention Bias）”注入后续 ViT 层，利用 LLM 的下游任务梯度回传，自动学习哪些空间区域对推理更重要。
    *   **时间维度：** 引入辅助损失（Auxiliary Loss），计算相邻帧对应 Patch 的余弦相似度。相似度越高，评分越低，从而识别并剔除时间上的冗余信息。
*   **高效打包算法（Packing Algorithm）：**
    *   由于不同帧的剪枝比例不同（非均匀剪枝），会产生稀疏张量。论文采用“首次适应递减算法（First-fit Descending）”将存活的 Token 重新打包成紧凑的稠密张量，从而在硬件层面实现真正的计算加速。
*   **端到端训练：** 整个模块与 VLM 主干网络共同训练，总损失函数为：$L = L_{task} + \lambda L_{sim}$，其中 $L_{sim}$ 是针对时间冗余的均方误差（MSE）损失。

*   **对比方法：** 
    *   **SOTA 模型：** Qwen3-VL-4B, InternVL3.5-8B, PLM-8B。
    *   **剪枝基准：** 随机剪枝（Random）、启发式剪枝（Heuristic）、Token Merging (ToMe)。

论文进行了较为充分的实验验证：
*   **多维度评估：** 涵盖了 13 个主流视频理解榜单。
*   **消融实验：** 探讨了剪枝层深度（$l$）、辅助损失的必要性、不同剪枝比例（50% 到 90%）的影响。

#### 6. 主要结论与发现
*   **高效性：** 剪掉 50% 的视觉 Token 后，模型在 13 个任务上的平均性能仅下降 **0.7%**，而计算效率提升了 **62%**。
*   **测试时缩放（TTS）：** 在推理阶段，通过剪枝节省的预算可以用来输入更多帧（如从 64 帧增加到 128 帧），在长视频任务上反而比未剪枝的基线提升了 **0.5-1%** 的性能。
*   **层深影响：** 在 ViT 极早期（如第 0 层）剪枝会显著损害性能，选择第 3 层是效率与效果的最佳平衡点。

#### 7. 优点
*   **简单且统一：** 无需复杂的文本-视觉交叉注意力机制，也无需复杂的 Token 合并逻辑，易于集成。
*   **硬件友好：** 通过打包算法解决了非均匀剪枝在 GPU 上难以加速的痛点。
*   **可解释性：** 可视化证明模型确实学会了关注语义关键区域（如游戏中的角色、人的面部）。
*   **泛化性：** 实验证明该方法在不损害图像任务性能的前提下，显著提升了视频处理效率。

#### 8. 不足与局限
*   **超参数依赖：** 剪枝比例 $k$ 是预设的固定值，无法根据视频内容的复杂程度动态调整每条样本的剪枝率。
*   **极端剪枝风险：** 当剪枝率超过 80% 时，性能下降开始变得明显，对于极细微动作的捕捉可能存在信息丢失。
*   **架构限制：** 虽然理论上通用，但实验主要基于 ViT + LLM 架构，对于其他类型的视觉编码器（如卷积网络）未经测试。
