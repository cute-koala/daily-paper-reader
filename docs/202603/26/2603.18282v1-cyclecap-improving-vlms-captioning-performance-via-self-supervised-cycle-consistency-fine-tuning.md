## 摘要
视觉语言模型（VLMs）在图像描述、视觉问答和视觉推理方面取得了显著进展。然而，它们仍容易出现视觉-语言失配问题，经常生成过于笼统或存在幻觉的描述。现有方法通常通过指令微调（需要昂贵的大规模标注数据集）或复杂的测试时框架（用于描述细化）来解决这一问题。在本研究中，我们从循环一致性（cycle consistency）的角度重新审视图像-文本对齐：给定一张图像和由图像转文本模型生成的描述，通过文本转图像模型的反向映射应能重建出一张与原图高度匹配的图像。在我们的设置中，VLM 作为图像转文本组件，而预训练的文本转图像模型通过根据生成的描述重建图像来闭合循环。基于此，我们提出了 CycleCap，这是一种利用群体相对策略优化（GRPO）来改进图像描述的微调方案，其奖励基于实时计算的原始图像与重建图像之间的相似度。与以往使用循环一致性损失构建偏好数据集的工作不同，我们的方法直接将循环一致性作为自监督训练信号。这使得仅使用原始图像成为可能，消除了对精选图像-文本数据集的需求，同时引导 VLM 生成更准确且有据可依（grounded）的文本描述。将 CycleCap 应用于参数量从 1B 到 7B 的四种 VLM，在图像描述和幻觉基准测试中均取得了持续的改进，超越了依赖监督循环一致性训练的最先进方法。

## Abstract
Visual-Language Models (VLMs) have achieved remarkable progress in image captioning, visual question answering, and visual reasoning. Yet they remain prone to vision-language misalignment, often producing overly generic or hallucinated descriptions. Existing approaches address this via instruction tuning-requiring costly, large-scale annotated datasets or via complex test-time frameworks for caption refinement. In this work, we revisit image-text alignment through the lens of cycle consistency: given an image and a caption generated by an image-to-text model, the backward mapping through a text-to-image model should reconstruct an image that closely matches the original. In our setup, a VLM serves as the image-to-text component, while a pre-trained text-to-image model closes the loop by reconstructing the image from the generated caption. Building on this, we introduce CycleCap, a fine-tuning scheme to improve image captioning using Group Relative Policy Optimization (GRPO) with a reward based on the similarity between the original and reconstructed images, computed on-the-fly. Unlike previous work that uses cycle consistency loss for preference dataset construction, our method leverages cycle consistency directly as a self-supervised training signal. This enables the use of raw images alone, eliminating the need for curated image-text datasets, while steering the VLM to produce more accurate and grounded text descriptions. Applied to four VLMs ranging from 1B to 7B parameters, CycleCap yields consistent improvements across captioning and hallucination benchmarks, surpassing state-of-the-art methods that rely on supervised cycle consistency training.

*   **核心问题**：视觉语言模型（VLMs）在生成图像描述（Captioning）时，常出现**视觉-语言失配**问题，表现为描述过于笼统、缺乏细节或产生“幻觉”（生成图中不存在的内容）。
*   **研究背景**：现有的改进方案通常依赖大规模人工标注的指令微调（成本极高）或复杂的推理时细化框架（计算开销大）。本文旨在探索一种**无需人工标注**、利用**循环一致性（Cycle Consistency）**原理的自监督微调方法。

### 2. 方法论
*   **核心思想**：借鉴翻译领域的循环一致性，即“图像 $\rightarrow$ 文本 $\rightarrow$ 图像”的闭环。如果 VLM 生成的描述足够准确，那么一个预训练的文本转图像（T2I）模型应该能根据该描述还原出与原图高度相似的图像。
*   **关键技术细节**：
    *   **循环一致性奖励（Cycle Consistency Reward）**：计算原始图像 $x$ 与重构图像 $G(F(x))$ 之间的相似度。论文主要采用 **DreamSim** 指标，因为它能更好地捕捉人类感知的语义和结构相似性。
    *   **GRPO 算法优化**：采用群体相对策略优化（Group Relative Policy Optimization）。对于每张图，VLM 生成一组（$n$个）候选描述，通过 T2I 模型重构图像并计算奖励，利用组内相对得分（Advantage）来更新模型参数。
    *   **训练流程**：冻结 T2I 模型（如 Stable Diffusion 3），仅通过 LoRA 插件微调 VLM。这种方式避免了传统强化学习（如 PPO）中复杂的 Critic 网络，显著降低了显存占用。

### 3. 实验设计
*   **数据集**：使用 **COCO 2014 训练集**（约 8.3 万张图像）进行微调，且**不使用**其自带的参考描述。

    *   **CapsBench**：通过 LLM 回答基于描述的“是/否”问题来测试视觉接地能力。
    *   **MMHal**：专门评估模型产生幻觉的程度。
*   **对比方法**：包括各模型的 Baseline、**RICO-Flash**（基于 GPT-4o 反馈的微调）和 **CyclePref**（基于大规模偏好数据集的 DPO 微调）。

### 6. 主要结论与发现
*   **性能提升**：CycleCap 在所有测试的 VLM 上均实现了持续的性能增长，尤其在对象识别、属性描述和减少幻觉方面表现突出。
*   **超越监督学习**：尽管是自监督方法，CycleCap 的表现超越了依赖 GPT-4o 标注或大规模偏好数据集的 SOTA 方法（如 RICO 和 CyclePref）。
*   **可扩展性**：随着 T2I 模型（如 FLUX）和感知指标的进步，CycleCap 的增益会进一步放大。

### 7. 优点
*   **完全自监督**：摆脱了对昂贵人工标注和闭源模型（如 GPT-4）API 的依赖，仅需原始图像即可训练。
*   **训练高效**：利用 GRPO 简化了强化学习流程，适合在有限算力下优化中大规模 VLM。
*   **描述质量高**：生成的描述不仅更详细，而且在空间布局和物体属性上与视觉内容高度对齐。

### 8. 不足与局限
*   **图像空间偏差**：奖励信号完全来自图像像素/特征对比，可能无法捕捉到文本逻辑中的细微错误（如语法或常识逻辑）。

*   **评估依赖**：部分评估基准依赖于 LLM-as-a-judge，可能存在模型偏见或对长文本的评分偏好。
