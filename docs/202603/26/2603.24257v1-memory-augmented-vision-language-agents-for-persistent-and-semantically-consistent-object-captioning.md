## 摘要
视觉-语言模型（VLM）在不同视角下对同一物体的描述往往不一致，这阻碍了具身智能体随时间构建一致语义表示的能力。以往的方法通过离线多视图聚合或将探索、数据关联和描述学习解耦的多阶段流水线来解决不一致性，但对先前观察到的物体的推理能力有限。在本文中，我们介绍了一种统一的、记忆增强的视觉-语言智能体，它在单个自回归框架内同时处理数据关联、物体描述和探索策略。该模型处理当前的 RGB 观测、自顶向下的探索地图以及序列化为物体级 token 的物体级情节记忆，从而确保在长序列中保持持久的物体身份和语义一致性。为了以自监督方式训练模型，我们利用基于分歧的策略和一种在多视图描述历史中强制执行一致性的伪描述模型，在逼真的 3D 环境中收集了一个数据集。在手动标注的物体级测试集上的广泛评估表明，与基准模型相比，标准描述分数提高了高达 11.86%，描述自相似性提高了 7.39%，同时通过紧凑的场景表示实现了可扩展的性能。代码、模型权重和数据可在 https://github.com/hsp-iit/epos-vlm 获取。

## Abstract
Vision-Language Models (VLMs) often yield inconsistent descriptions of the same object across viewpoints, hindering the ability of embodied agents to construct consistent semantic representations over time. Previous methods resolved inconsistencies using offline multi-view aggregation or multi-stage pipelines that decouple exploration, data association, and caption learning, with limited capacity to reason over previously observed objects. In this paper, we introduce a unified, memory-augmented Vision-Language agent that simultaneously handles data association, object captioning, and exploration policy within a single autoregressive framework. The model processes the current RGB observation, a top-down explored map, and an object-level episodic memory serialized into object-level tokens, ensuring persistent object identity and semantic consistency across extended sequences. To train the model in a self-supervised manner, we collect a dataset in photorealistic 3D environments using a disagreement-based policy and a pseudo-captioning model that enforces consistency across multi-view caption histories. Extensive evaluation on a manually annotated object-level test set, demonstrate improvements of up to +11.86% in standard captioning scores and +7.39% in caption self-similarity over baseline models, while enabling scalable performance through a compact scene representation. Code, model weights, and data are available at https://github.com/hsp-iit/epos-vlm

以下是对该论文的结构化总结：

### 1. 核心问题与研究动机
*   **核心问题**：现有的视觉-语言模型（VLM）主要在静态图像上训练。当具身智能体在 3D 空间移动时，由于视角、距离和遮挡的变化，VLM 往往会对同一个物理物体给出矛盾的描述（例如：从远处看是“沙发”，近处看变成“床”）。这种“语义漂移”破坏了物体的身份持久性，阻碍了智能体进行长程推理和可靠建图。
*   **研究背景**：先前的方法通常将探索、数据关联（确定当前看到的物体是否见过）和描述生成解耦为多个独立阶段，或者依赖离线的后处理聚合，缺乏实时调整探索策略以消除语义歧义的能力。

### 2. 方法论
*   **核心思想**：将数据关联、物体描述和导航动作集成到一个统一的自回归 Transformer 框架中，并引入“物体级情节记忆”作为上下文。
*   **关键技术细节**：
    *   **记忆令牌化（Memory Tokenization）**：将情节记忆（包含物体 ID、3D 坐标、历史描述及其频率）序列化为文本 token，直接输入 VLM。这使得预训练模型能像阅读文本一样“阅读”物体的历史状态。
    *   **统一输出序列**：模型在每个时间步输出一个结构化序列：`[MATCH]`（将当前检测到的物体关联到记忆 ID 或标记为新物体）、`[CAPTION]`（生成一致的描述）和 `[ACTION]`（决定下一步动作）。
    *   **3D-CPS 伪标签生成器**：为了进行自监督训练，作者开发了一个 3D 感知的伪描述模型，它通过聚合多视角下的观察，过滤掉视角相关的噪声，生成一个物体固有的、一致的“真值”描述。
    *   **主动探索策略**：模型被训练为倾向于重新访问那些语义描述不确定（分歧大）的物体，通过获取更多视角来消除歧义。

### 3. 实验设计
*   **数据集与场景**：在 **Habitat** 仿真平台中进行，使用了 **HM3D**（128 个训练场景，28 个测试场景）和 **Gibson**（用于跨领域泛化测试，32 个场景）。
*   **Benchmark（基准指标）**：
    *   **描述准确性**：BLEU-4, METEOR, ROUGE-L, CIDEr, SPICE。
    *   **语义一致性**：使用 SBERT 嵌入的余弦相似度（CS）和四分位距（IQR）来衡量同一物体在不同视角下描述的相似程度。
    *   **数据关联**：跟踪准确率（Acc）、身份切换（IDSW）等。
*   **对比方法**：
    *   **基础 VLM**：Qwen3-VL-2B, BLIP-2, InternVL-3B, CoCa, Florence。
    *   **多阶段流水线**：如 IC3, ECO-score, 以及作者之前的 LD-CPS。

### 4. 资源与算力
*   **硬件**：使用了 **2 张 NVIDIA A100 GPU**。
*   **训练细节**：
    *   采用 **LoRA**（低秩自适应）进行高效微调。
    *   训练 3 个 epoch，batch size 为 16，使用 AdamW 优化器。
    *   微调的基础模型是 Qwen3-VL-2B。

### 5. 实验数量与充分性
*   **实验规模**：论文涵盖了伪描述生成质量评估、物体级描述准确性对比、跨视角一致性分析、数据关联性能测试、计算扩展性（推理时间与内存）分析以及动作策略的消融实验。
*   **充分性评价**：实验设计较为充分。作者不仅在同分布的 HM3D 上测试，还专门在 **Gibson 数据集上进行了零样本（Zero-shot）泛化实验**，证明了模型的鲁棒性。此外，通过消融实验（如去掉记忆模块）明确验证了记忆机制的贡献。

### 6. 主要结论与发现
*   **性能提升**：EPOS-VLM 在 SPICE 指标上比最强的基准 VLM 提升了 **11.86%**，在描述一致性（Mean CS）上提升了 **7.39%**。
*   **记忆的重要性**：消融实验显示，去掉情节记忆会导致描述准确性大幅下降，证明了长程语义推理依赖于持久化存储而非瞬时视觉。
*   **高效性**：相比于基于稠密点云的存储方法，EPOS-VLM 的文本化记忆极其节省空间（内存占用从 GB 级降至 KB 级），且推理时间不随探索时长线性增长。

### 7. 优点（亮点）
*   **端到端集成**：首次将具身导航中的“看、想、动、记”四个环节统一在一个自回归序列生成任务中。
*   **主动消除歧义**：模型能够根据语义不确定性主动选择观察视角，这比随机探索或单纯的几何探索更智能。
*   **可扩展性**：通过将 3D 空间信息和语义历史转化为 token，巧妙地利用了 LLM 的上下文处理能力，避开了复杂的 3D 几何计算。

### 8. 不足与局限
*   **依赖外部分割**：模型仍需依赖预训练的实例分割模型（如 Mask2Former），如果分割出错，后续的数据关联和描述也会受影响。
*   **静态环境假设**：实验主要在静态 3D 场景中进行，未讨论在物体会移动或环境发生动态变化时的表现。
*   **传感器噪声**：实验基于仿真环境的理想传感器，在现实世界中，深度图噪声和定位误差可能会降低 3D 坐标匹配的准确性。
