## 摘要
组合图像检索（CIR）旨在根据参考图像和修改文本检索目标图像。然而，现有方法往往难以从参考图像中提取出在文本修改提示下最能反映用户意图的正确语义线索，从而导致受到无关视觉噪声的干扰。在本文中，我们提出了一种用于 CIR 的新型基于多模态思维链推理的多层级视觉选择（MCoT-MVS）方法，该方法集成了由多模态大语言模型（MLLM）推理线索引导的注意力感知多层级视觉特征。具体而言，我们利用 MLLM 对多模态组合输入进行思维链推理，生成保留、移除和目标推断的文本。随后，这些文本线索引导两个参考视觉注意力选择模块，从参考图像中选择性地提取具有辨别力的补丁级（patch-level）和实例级（instance-level）语义。最后，为了有效地将这些多粒度视觉线索与修改文本及想象的目标描述进行融合，我们设计了一个加权分层组合模块，在统一的嵌入空间中将组合查询与目标图像进行对齐。在 CIRR 和 FashionIQ 两个 CIR 基准测试上的大量实验表明，我们的方法始终优于现有方法，并达到了新的最先进性能。代码和训练模型已公开发布。

## Abstract
Composed Image Retrieval (CIR) aims to retrieve target images based on a reference image and modified texts. However, existing methods often struggle to extract the correct semantic cues from the reference image that best reflect the user's intent under textual modification prompts, resulting in interference from irrelevant visual noise. In this paper, we propose a novel Multi-level Vision Selection by Multi-modal Chain-of-Thought Reasoning (MCoT-MVS) for CIR, integrating attention-aware multi-level vision features guided by reasoning cues from a multi-modal large language model (MLLM). Specifically, we leverage an MLLM to perform chain-of-thought reasoning on the multimodal composed input, generating the retained, removed, and target-inferred texts. These textual cues subsequently guide two reference visual attention selection modules to selectively extract discriminative patch-level and instance-level semantics from the reference image. Finally, to effectively fuse these multi-granular visual cues with the modified text and the imagined target description, we design a weighted hierarchical combination module to align the composed query with target images in a unified embedding space. Extensive experiments on two CIR benchmarks, namely CIRR and FashionIQ, demonstrate that our approach consistently outperforms existing methods and achieves new state-of-the-art performance. Code and trained models are publicly released.

*   **核心问题**：组合图像检索（CIR）要求根据“参考图像”和“修改文本”检索目标图像。现有方法在处理复杂的修改指令时，难以准确识别参考图像中哪些部分应保留、哪些应删除，导致检索向量中混入大量无关的视觉噪声（如本应被替换的背景或物体）。
*   **研究动机**：作者认为，现有的注意力机制多基于浅层的图文相关性，缺乏显式的语义推理。通过引入多模态大语言模型（MLLM）的思维链（CoT）推理能力，可以显式地分解用户意图，从而指导模型在不同粒度上精确选择视觉特征。

### 2. 方法论
MCoT-MVS 框架主要由三个核心阶段组成：
*   **多模态思维链推理（MCoT Reasoning）**：
    *   利用预训练 MLLM（如 Qwen2.5-VL）对输入进行四步推理：描述参考图、解释修改指令、推断目标图描述、识别需“保留”和“删除”的视觉元素。
    *   输出三种文本线索：保留文本（$RT$）、删除文本（$DT$）和推断的目标文本（$TT$）。
*   **多层级视觉参考选择（Multi-level Vision Selection）**：
    *   **补丁级选择（PVRS）**：计算 CLIP-ViT 提取的图像补丁（Patch）与 $RT$ 和 $DT$ 的余弦相似度，通过“保留权重减去删除权重”的方式对补丁进行对比重加权，抑制噪声。
    *   **实例级选择（IVRS）**：利用 Grounded-SAM 提取参考图中的具体实例（Instance），在更高层级的语义空间内根据推理线索筛选关键物体。
*   **加权分层组合模块（WHC）**：
    *   设计了一个自适应的 Combiner 结构，将筛选后的补丁特征、实例特征与原始修改文本、推断目标文本进行分层融合。通过学习不同特征的贡献权重，生成最终的统一查询向量。

### 6. 主要结论与发现
*   **性能突破**：MCoT-MVS 在 CIRR 和 FashionIQ 上均刷新了 SOTA 记录。在 CIRR 上，平均 Recall 提升至 82.52%；在 FashionIQ 上，平均 R@10 达到 63.24%。
*   **推理的重要性**：显式的“保留/删除”推理比直接融合原始图文信息更有效，能显著提升模型对细微修改意图的捕捉能力。
*   **多级协同**：补丁级特征提供细粒度背景信息，实例级特征提供明确的物体语义，两者互补。

### 7. 优点与亮点
*   **显式意图分解**：首次通过 MLLM 显式生成“保留”和“删除”清单来指导视觉特征选择，增强了模型的可解释性。
*   **多粒度视觉处理**：结合了 Patch 和 Instance 两个层面的视觉筛选，既照顾了全局背景也突出了核心物体。
*   **即插即用潜力**：其推理和选择逻辑可以较容易地适配到其他基于 CLIP 的检索框架中。

### 8. 不足与局限
*   **推理延迟**：依赖 MLLM 生成思维链，虽然效果好，但在实时检索场景下（如毫秒级响应需求）可能存在延迟瓶颈。
*   **幻觉风险**：MLLM 在推理目标文本时可能产生“幻觉”（Hallucination），虽然 WHC 模块有加权过滤，但错误推理仍可能干扰最终检索。
*   **流程复杂性**：集成了 MLLM、Grounded-SAM 和 CLIP 多个大模型，系统复杂度较高，部署成本相对较大。
