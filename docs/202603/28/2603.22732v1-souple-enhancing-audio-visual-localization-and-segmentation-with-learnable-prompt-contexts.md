## 摘要
大规模预训练图文模型展现出鲁棒的多模态表示，但将对比语言-图像预训练（CLIP）模型应用于视听定位仍具挑战性。用音频嵌入标记（[V_A]）替换分类标记（[CLS]）难以捕捉语义线索，且提示词“a photo of a [V_A]”无法在音频嵌入与上下文标记之间建立有意义的联系。为解决这些问题，我们提出了声学感知提示学习（SOUPLE），该方法用可学习的上下文标记替换了固定提示。这些标记结合了视觉特征，为掩码解码器生成条件上下文，从而有效地桥接了音频和视觉输入之间的语义对应关系。在 VGGSound、SoundNet 和 AVSBench 上的实验表明，SOUPLE 提升了定位与分割性能。

## Abstract
Large-scale pre-trained image-text models exhibit robust multimodal representations, yet applying the Contrastive Language-Image Pre-training (CLIP) model to audio-visual localization remains challenging. Replacing the classification token ([CLS]) with an audio-embedded token ([V_A]) struggles to capture semantic cues, and the prompt "a photo of a [V_A]" fails to establish meaningful connections between audio embeddings and context tokens. To address these issues, we propose Sound-aware Prompt Learning (SOUPLE), which replaces fixed prompts with learnable context tokens. These tokens incorporate visual features to generate conditional context for a mask decoder, effectively bridging semantic correspondence between audio and visual inputs. Experiments on VGGSound, SoundNet, and AVSBench demonstrate that SOUPLE improves localization and segmentation performance.

*   **核心问题**：如何有效地将预训练的 CLIP（对比语言-图像预训练）模型应用于视听定位（Audio-Visual Localization）任务。
*   **研究背景**：现有的 CLIP 适配方法（如 ACL-SSL）通常将文本分类标记 `[CLS]` 替换为音频嵌入标记 `[VA]`，并配合固定提示词（如 "a photo of a [VA]"）。

    1.  固定短语（如 "a photo of a"）与音频标记 `[VA]` 之间缺乏语义关联。
    2.  这种静态提示无法根据具体的视觉内容捕捉动态的语义线索，限制了模型在处理未见过的对象或复杂场景时的泛化能力。

### 2. 论文提出的方法论
*   **核心思想**：提出 **SOUPLE**（声学感知提示学习）框架，将手工设计的固定提示词替换为**可学习的、实例条件化的上下文标记**。
*   **关键技术细节**：
    *   **Meta-net（元网络）**：一个轻量级的非线性瓶颈结构（Linear-ReLU-Linear），它接收图像特征 $F_I$，并生成 $M$ 个可学习的上下文标记 $[V_1][V_2]...[V_M]$。这使得提示词能够根据输入图像的内容动态调整。
    *   **音频分词器（Audio Tokenizer）**：将音频特征转换为与 CLIP 文本编码器兼容的音频嵌入标记 $[V_A]$。

    *   **掩码解码器（Mask Decoder）**：结合图像特征和视听文本特征，生成最终的分割掩码。
*   **算法流程**：
    1.  提取图像特征并经 Meta-net 生成动态上下文标记。
