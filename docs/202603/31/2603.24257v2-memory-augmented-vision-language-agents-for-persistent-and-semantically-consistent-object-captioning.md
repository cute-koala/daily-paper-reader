## 摘要
视觉-语言模型（VLM）在不同视角下对同一物体的描述往往不一致，这阻碍了具身智能体随时间构建一致语义表示的能力。以往的方法通过离线多视图聚合或将探索、数据关联和描述学习解耦的多阶段流水线来解决不一致性，但对先前观察到的物体的推理能力有限。在本文中，我们提出了一种统一的、记忆增强的视觉-语言智能体，它在单个自回归框架内同时处理数据关联、物体描述和探索策略。该模型处理当前的 RGB 观测、自顶向下的探索地图以及序列化为物体级 token 的物体级情境记忆，从而确保在长序列中物体身份的持久性和语义一致性。为了以自监督方式训练模型，我们利用基于分歧的策略和在多视图描述历史间强制一致性的伪描述模型，在逼真的 3D 环境中收集了一个数据集。在人工标注的物体级测试集上的广泛评估表明，与基准模型相比，该方法在标准描述分数上提升了高达 11.86%，在描述自相似性上提升了 7.39%，同时通过紧凑的场景表示实现了可扩展的性能。代码、模型权重和数据可在 https://hsp-iit.github.io/epos-vlm/ 获取。

## Abstract
Vision-Language Models (VLMs) often yield inconsistent descriptions of the same object across viewpoints, hindering the ability of embodied agents to construct consistent semantic representations over time. Previous methods resolved inconsistencies using offline multi-view aggregation or multi-stage pipelines that decouple exploration, data association, and caption learning, with limited capacity to reason over previously observed objects. In this paper, we introduce a unified, memory-augmented Vision-Language agent that simultaneously handles data association, object captioning, and exploration policy within a single autoregressive framework. The model processes the current RGB observation, a top-down explored map, and an object-level episodic memory serialized into object-level tokens, ensuring persistent object identity and semantic consistency across extended sequences. To train the model in a self-supervised manner, we collect a dataset in photorealistic 3D environments using a disagreement-based policy and a pseudo-captioning model that enforces consistency across multi-view caption histories. Extensive evaluation on a manually annotated object-level test set, demonstrate improvements of up to +11.86% in standard captioning scores and +7.39% in caption self-similarity over baseline models, while enabling scalable performance through a compact scene representation. Code, model weights, and data are available at https://hsp-iit.github.io/epos-vlm/.

### 1. 核心问题与整体含义（研究动机和背景）
*   **核心问题**：现有的视觉-语言模型（VLM）在具身智能体（Embodied Agents）应用中存在**语义不一致性**。当智能体从不同视角、距离或遮挡条件下观察同一物体时，VLM 往往会给出矛盾的描述（例如：一会儿说是“沙发”，一会儿说是“扶手椅”）。
*   **研究背景**：这种“语义漂移”破坏了物体的身份持久性，阻碍了智能体进行长期的语义建图和推理。以往的方法通常将探索、数据关联和描述生成解耦为多个独立阶段，缺乏实时利用历史记忆来纠正当前感知偏差的能力。
*   **整体含义**：本文旨在构建一个统一的智能体框架，通过引入物体级情节记忆，使智能体能够像人类一样，通过多视角观察和记忆积累，构建稳定且一致的物体语义理解。

### 2. 方法论
*   **核心思想**：提出 **EPOS-VLM** 框架，将**数据关联（Data Association）**、**物体描述（Object Captioning）**和**导航动作预测（Action Prediction）**集成到一个统一的自回归视觉-语言-动作（VLA）模型中。
*   **关键技术细节**：
    *   **物体级情节记忆（Episodic Object Memory）**：维护一个结构化数据库，记录每个物体的持久 ID、3D 坐标位置以及历史描述频率。
    *   **结构化 Token 化**：将上述记忆序列化为特定的文本 Token（如 `[OBJ]`, `[POS]`, `[CAPTIONS]`），直接输入预训练 VLM 的上下文窗口，使其具备长程推理能力。
    *   **统一自回归输出**：模型在每个时间步处理当前 RGB 图像、探索地图和记忆 Token，依次输出：
        1.  `[MATCH]`：将当前检测到的物体与记忆中的 ID 关联或标记为新物体。
        2.  `[CAPTION]`：基于记忆修正后的物体描述。
        3.  `[ACTION]`：下一步导航动作（如前进、左转）。
    *   **3D-CPS 伪标签生成器**：在训练阶段，利用 3D 几何信息聚合多视角观察，生成高质量、视角无关的物体描述作为自监督学习的监督信号。
    *   **基于分歧的探索策略**：智能体会主动导航至那些描述不一致（语义分歧大）的区域，以获取更多信息来消除歧义。

### 3. 实验设计
*   **数据集与场景**：在 **Habitat** 仿真平台中进行，使用 **HM3D** 数据集进行训练和验证，使用 **Gibson** 数据集评估跨环境的泛化能力。
*   **Benchmark**：构建了一个手动标注的物体级描述测试集。
*   **对比方法**：
    *   **基础 VLM**：Qwen3-VL, BLIP-2, InternVL, CoCa, Florence。
    *   **聚合方法**：IC3, ECO-score, LD-CPS。
    *   **消融实验**：对比了移除记忆模块（w/o memory）的版本，以及不同的探索策略（随机目标、前沿探索）。

### 4. 资源与算力
*   **硬件**：使用了 **2 张 NVIDIA A100 GPU**。
*   **训练细节**：
    *   采用 **LoRA**（低秩自适应）进行高效微调。
    *   训练 3 个 Epoch，使用 AdamW 优化器。
    *   Batch size 为 16（每张卡），梯度累积步数为 4。
    *   基础模型为 Qwen3-VL-2B。

### 5. 实验数量与充分性
*   **实验规模**：涵盖了伪描述准确性、物体描述精度、跨视角语义一致性、数据关联准确率、计算可扩展性（推理时间与内存占用）以及动作策略影响等多个维度。
*   **充分性评价**：实验设计非常全面。不仅对比了最先进的 VLM 模型，还通过 Gibson 数据集验证了**泛化性**。通过消融实验明确了记忆模块的贡献。手动标注测试集确保了评估的**客观性**。此外，对推理效率的分析证明了该方法在具身设备上运行的可行性。

### 6. 主要结论与发现
*   **性能提升**：EPOS-VLM 在标准描述指标（如 SPICE）上比最强基准提升了 **11.86%**，在语义一致性（Cosine Similarity）上提升了 **7.39%**。
*   **记忆的重要性**：移除记忆后，模型性能大幅下降，证明了持久化物体表示是实现一致性的关键。
*   **主动探索的价值**：相比于单纯的几何覆盖探索，基于语义分歧的探索策略能显著提高描述的准确度。
*   **高效性**：相比于基于稠密点云的记忆方法，本文的 Token 化稀疏记忆将内存占用从 GB 级降低到 KB 级，且推理时间保持恒定，不随步数增加而线性增长。

### 7. 优点
*   **架构统一**：将感知、记忆、关联和行动整合进一个 Transformer 序列，避免了多阶段误差累积。
*   **语义持久性**：通过记忆增强，成功解决了 VLM 在动态导航中的“健忘”和“矛盾”问题。
*   **可扩展性强**：文本化的记忆表示极其节省资源，适合长程任务。
*   **自监督闭环**：利用 3D 几何自动生成高质量训练数据，降低了对人工标注的依赖。

### 8. 不足与局限
*   **依赖外部分割模型**：目前仍依赖预训练的实例分割模型（如 Mask2Former），若分割出错会影响后续关联和描述。
*   **静态环境假设**：实验主要在静态 3D 场景中进行，尚未充分探讨在物体会移动的动态环境中的表现。
*   **传感器噪声**：实验基于仿真环境的理想深度和位姿，现实世界中的传感器噪声可能会降低 3D 投影和数据关联的鲁棒性。
