## 摘要
视频推理分割要求在复杂自然语言查询下，在数百帧中实现像素级精确的目标跟踪，从而产生密集的时空标记，其二次自注意力成本使得长视频处理变得不可行。现有方法通过标记压缩来解决这一问题，但通常作用于缺乏时间上下文的编码器特征，在内容冗余得以可靠评估之前限制了选择。知情压缩需要上下文感知，但在全分辨率下获取这种感知会产生压缩旨在减少的相同二次成本。状态空间模型解决了这一约束，因为其线性递归以$\mathcal{O}(T)$成本在时间上下文上条件性地选择每个标记，产生可评估内容冗余的表示。在此基础上，选择性时空聚合与压缩（STAC）通过解耦的双向空间和因果时间扫描丰富特征，利用递归衍生的冗余进行分层压缩，并使用分割目标优化的自适应阈值。STAC在零样本流式兼容设置下，在推理分割基准上实现了85%的标记减少和1.8倍加速，同时超越了无压缩基线。代码可在\href{https://github.com/MCG-NKU/nku-video}{此处}获取。

## Abstract
Video reasoning segmentation demands pixel-accurate object tracking across hundreds of frames under complex natural language queries, producing dense spatiotemporal tokens whose quadratic self-attention cost makes long-video processing prohibitive. Existing methods address this through token compression, yet typically operate on encoder features lacking temporal context, constraining selection before content redundancy can be reliably assessed. Informed compression requires contextual awareness, but acquiring that awareness at full resolution incurs the same quadratic cost compression aims to reduce. State-space models resolve this constraint, as their linear recurrence selectively conditions each token on temporal context at $\mathcal{O}(T)$ cost, producing representations where content redundancy becomes assessable. Building on this, Selective SpatioTemporal Aggregation and Compression (STAC) enriches features via decoupled bidirectional spatial and causal temporal scanning, leveraging recurrence-derived redundancy for hierarchical compression with adaptive thresholds optimised with segmentation objective. STAC achieves 85% token reduction and 1.8$\times$ speedup while surpassing compression-free baselines on reasoning segmentation benchmarks in a zero-shot streaming-compatible setting. ## 1. 论文的核心问题与整体含义（研究动机和背景）

- **问题**：视频推理分割需要在复杂自然语言查询下，对长视频（数百帧）进行像素级目标跟踪，产生大量时空token。标准的自注意力机制具有二次复杂度（O(T²)），导致长视频处理成本极高，甚至超出大多数MLLM的上下文窗口（如32K）。
- **现有方法局限**：
  - 帧选择或空间压缩等方法缺乏跨帧的时序上下文，在冗余评估前就做了压缩决策，导致关键运动信息丢失。
  - 获取时序上下文需要全分辨率处理，恰恰又带来二次成本，形成“压缩与理解”之间的张力。
- **核心洞察**：状态空间模型（SSM，如Mamba）具有线性递归特性（O(T)成本），能对每个token基于时序上下文进行选择性条件化，从而在特征中暴露内容冗余。STAC利用这一信号实现原理性的视频token压缩。

## 2. 论文提出的方法论

### 核心思想
- **先丰富后压缩**：在压缩之前先通过SSM对编码器特征进行时空上下文增强，使冗余信号成为可评估的语义特征，再基于此进行分层压缩。
- **解耦空间与时间扫描**：空间维度采用双向扫描（非因果），时间维度采用因果扫描（单向），既保证了空间语义完整性，又支持流式在线推理。
- **任务导向的差分压缩**：通过直通估计器将分割梯度反向传播到压缩决策中，使token保留策略直接优化分割精度。

### 关键技术细节
1. **状态信息时空聚合器（SSA）**：
   - **空间聚合**：每帧独立进行双向Mamba扫描，取前向和后向平均，消除方向偏差。
   - **时间聚合**：在空间维度上并行进行因果Mamba扫描，更新隐藏状态$h_{n,t} = \bar{A}_t h_{n,t-1} + \bar{B}_t F_{\text{spatial}}[t,n]$，保持线性复杂度且支持流式处理。
2. **分层状态自适应压缩（HSC）**：
   - **自适应时间压缩**：计算帧间相似度$s_t$，利用指数移动平均（EMA）估计在线统计量$\mu_t, \sigma_t$，设定自适应阈值$\tau_t = \mu_t + k\sigma_t$（k由轻量MLP学习），将相似帧合并。
   - **自适应空间压缩**：在时间压缩后的序列上计算各空间位置跨帧一致性$c_n$，同样用EMA阈值决定是否合并静态区域的token。整个流程先时后空，避免过早空间池化破坏运动信息。
3. **任务导向差分压缩**：
   - 总损失$\mathcal{L}_{\text{total}} = \mathcal{L}_{\text{seg}} + \lambda \mathcal{L}_{\text{comp}}$，其中$\mathcal{L}_{\text{seg}}$为交叉熵+Dice，$\lambda=0.001$。
   - 压缩正则项$\mathcal{L}_{\text{comp}}$根据视频复杂度动态调整：$ \mathcal{L}_{\text{comp}} = \frac{1}{B}\sum_{i=1}^B (1-\hat{C}_i)\cdot \frac{c_i}{\sqrt{T_i}}$，$\hat{C}_i$基于特征方差估计复杂度，平方根归一化使长视频惩罚递减。
   - 阈值参数通过直通估计器接收分割梯度更新。

## 6. 论文的主要结论与发现
- STAC利用SSM递归固有的冗余信号，实现了线性复杂度的原理性视频token压缩，在85% token缩减和1.8倍加速下，推理分割性能甚至超过无压缩基线。

- 分层压缩（先时后空）显著优于先空后时，因为过早空间池化会破坏运动信息。

## 7. 优点
- **创新性**：首次将SSM递归冗余信号用于视频token压缩，提出先富化后压缩的范式，克服了“先压缩后理解”的根本局限。
- **实用性**：因果时间设计支持流式在线部署，且压缩率自适应内容动态（静态区域>90%，动态区域30-50%）。
- **任务协同**：通过直通估计器实现端到端优化，压缩策略直接受分割目标引导。
- **零样本能力强**：仅用指代数据训练即可在推理任务上超越完全监督的压缩方法。
- **效率高**：线性复杂度，训练仅需2×A40和30小时，资源友好。

- **推理任务性能略低于最佳无压缩方法**（如MeViS、ReVOS上GLUS更高），说明压缩在运动密集场景仍有信息损失。
