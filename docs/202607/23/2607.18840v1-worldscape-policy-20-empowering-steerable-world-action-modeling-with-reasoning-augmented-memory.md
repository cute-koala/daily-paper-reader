## 摘要
世界动作模型（WAM）通过联合建模视觉状态转换和机器人动作，为机器人操作提供了一种有前景的范式。然而，现有的WAM受限于有限的时间上下文、粗粒度的剧集级语言监督以及主要是纯文本的条件设置，这阻碍了任务进度跟踪和细粒度语言-视频-动作的关联，同时限制了视觉上下文推理和跨具身迁移。在本文中，我们引入了WorldScape Policy 2.0，一种具有推理增强长短时记忆的可控WAM。其因果短时视觉记忆提供最近的观测作为DiT预填充，以保留局部交互动态，而其长短时事件记忆将历史VLM输出组织为全局历史、局部活跃和事件边界表示，用于进度感知检索。检索到的历史增强了感知和自回归生成的规划标记，为自主规划提供了隐式子目标条件；语义强制进一步将事件级指令语义转移至这一潜在规划通路。为建立细粒度多模态可控性，我们构建了ManipEvent-5M，一个事件基础的具身预训练数据集，包含近500万个事件段，带有对齐的动作轨迹、剧集级任务指令、段级子任务描述、目标图像和视频演示。这些设计为从高级指令进行自主规划和从细粒度文本、目标图像或视频上下文提示进行可控执行提供了统一接口。在仿真和真实世界平台上的实验展示了在长时域自主规划、细粒度指令跟随和上下文适应方面的卓越能力。

## Abstract
World Action Models (WAMs) offer a promising paradigm for robotic manipulation by jointly modeling visual state transitions and robot actions. However, existing WAMs are constrained by limited temporal context, coarse episode-level language supervision, and predominantly text-only conditioning, which hinder task-progress tracking and fine-grained language-video-action grounding while limiting visual-context reasoning and cross-embodiment transfer. In this paper, we introduce WorldScape Policy 2.0, a controllable WAM with reasoning-augmented long short-term memory. Its causal short-term visual memory supplies recent observations as DiT prefill to preserve local interaction dynamics, while its long short-term event memory organizes historical VLM outputs into global-history, local-active, and event-boundary representations for progress-aware retrieval. The retrieved history augments perception and autoregressively generated planning tokens, yielding an implicit subgoal condition for autonomous planning; semantic forcing further transfers event-level instruction semantics into this latent planning pathway. To establish fine-grained multimodal controllability, we construct ManipEvent-5M, an event-grounded embodied pretraining dataset containing nearly 5 million event segments with aligned action trajectories, episode-level task instructions, segment-level subtask captions, goal images, and video demonstrations. These designs provide a unified interface for autonomous planning from high-level instructions and controllable execution from fine-grained text, goal-image, or video-context prompts. Experiments in both simulation and real-world platforms demonstrate superior capabilities in long-horizon autonomous planning, fine-grained instruction following and in-context adaptation.

# WorldScape Policy 2.0 论文总结

## 1. 核心问题与整体含义（研究动机和背景）

- **研究痛点**：现有的世界动作模型（WAM）在机器人操作中存在三个关键限制：
  - **时间上下文有限**：大多数方法仅依赖当前观测或短历史窗口，缺乏**进度感知记忆**，导致不同阶段视觉相似时无法区分任务进度（例如桌面在子目标完成前后几乎一样，但下一步动作取决于已完成内容）。
  - **粗粒度语言-视频-动作对齐**：许多数据集同一 episode 的不同 segment 共享相同的任务级指令，导致模型学习视觉捷径而非真正理解细粒度动作意图。
  - **交互接口受限**：现有 WAM 通常仅支持文本控制，无法自然利用目标图像、演示视频或跨具身示例进行条件控制，限制了视觉上下文推理和跨具身迁移。

- **整体目标**：构建一个**可控的、具有推理增强长短期记忆**的世界动作模型，能够同时支持：
  - 从高层指令进行自主规划
  - 从细粒度文本、目标图像或视频上下文进行可控执行
  - 在长时域任务中跟踪进度并进行隐式子目标规划

## 2. 方法论：核心思想、关键技术细节

### 2.1 核心思想

- **关键洞见**：长程可控性需要两种互补的时间上下文：
  - **语义事件记忆**：用于推理任务进度
  - **帧级视觉记忆**：用于保留局部交互动态
- 将这两种记忆耦合在统一的世界动作模型中，通过**推理增强的长短期记忆**（Reasoning-Augmented Long Short-Term Memory）实现。

### 2.2 总体架构

- **两分支结构**：
  - **VLM 分支**：处理长短期事件记忆，进行进度感知检索和隐式子目标推理
  - **因果 DiT 分支**：处理短期视觉记忆（最近观测的 VAE 潜变量预填充）和可选的视觉提示（目标图像/演示视频）
- **统一接口**：支持三种交互模式：
  - 自主规划（仅接收 episode 级高层指令）
  - 细粒度指令跟随（接收 segment 级子任务描述）
  - 视觉提示适应（目标图像或视频演示）

### 2.3 关键技术细节

#### (a) 事件级世界动作建模（ManipEvent-5M 数据集驱动）
- 每个 episode 被分解为有序事件序列：`τ = {(观测, 动作, episode指令, 事件描述, 目标图像, 视频演示)}`
- 三种语言条件模式：
  - **细粒度指令**：T5 编码事件级描述，直接注入 DiT 交叉注意力
  - **自主规划**：VLM 分支根据高层指令和事件记忆生成隐式子目标条件
  - **视觉提示**：目标图像或视频演示作为持久化视觉预填充

#### (b) 推理增强的长短期记忆
- **短期视觉记忆**：保持最近 S_v 个观测的 VAE 潜变量，作为 DiT 因果预填充，提供局部运动连续性
- **长短期事件记忆**：
  - 维护历史 VLM 输出队列 Q_t（包含感知 token 和规划 token）
  - 构造三种记忆视图：
    - **全局历史隐变量**：对整个轨迹进行压缩池化
    - **局部活跃隐变量**：最近 S_e 个已完成的 chunk
    - **事件边界隐变量**：根据相邻 chunk 的语义变化（余弦距离）选择 Top-K 高变化点（作为锚点保留完整 token）
  - 检索与门控融合：当前 VLM token 作为 query，对三种视图和压缩历史进行交叉注意力，再通过门控机制选择性注入

#### (c) 隐式子目标潜在规划（Semantic Forcing）
- 训练时：使用细粒度事件描述（T5 嵌入）作为语义目标，通过余弦相似度损失（L_sem）将 VLM 推理潜变量拉向该语义空间（T5 编码器固定，stop-gradient）
- 推理时：无细粒度描述输入，模型仅依赖高层指令和事件记忆自主推断子目标

#### (d) 统一动作表示与具身适配器
- 动作表示：每手臂 10 维（delta 位置 3D + 6D 相对旋转 + 1D 夹爪），双20D
- 每个具身有专属动作编码器/解码器，DiT 主干共享

#### (e) 三阶段训练课程
1. **Stage 1**：事件级多模态 WAM 预训练（L_act + L_world），使用 ManipEvent-5M，激活短期视觉记忆，无事件记忆
2. **Stage 2**：记忆感知中训练（L_act + L_world + L_sem），引入 VLM 规划分支和长短期事件记忆，语义强制
3. **Stage 3**：下游交互后训练，针对具体任务适配
