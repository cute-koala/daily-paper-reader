## 摘要
近年来，恶劣天气条件下的域自适应视觉目标跟踪引起了广泛关注。尽管现有方法性能出色，但它们严重依赖于来自源域和目标域的大规模视频帧，这在源数据不可用的严格资源约束下不切实际。为克服这一限制，我们提出SFDATrack，一种广义的无源域自适应跟踪器，仅利用目标域中的恶劣天气样本进行鲁棒的状态估计。具体而言，SFDATrack首先采用带有双交互曼巴（DIM）块的平均教师骨干网络，从分类和增强样本中提取对天气变化具有鲁棒性的候选目标标记。随后，我们引入超球面原型投影（HPP）模块，将这些标记投影到潜在超球空间中的多域原型上。通过强制多域原型的域特定和域不变属性，SFDATrack能够无缝适应各种天气条件，并具有强大的泛化能力。在各种基准上的大量实验表明，SFDATrack相比最先进的方法取得了优越的性能。代码可在 https://github.com/watcherBR0/sfdatrack 获取。

## Abstract
Domain adaptive visual object tracking under adverse weather conditions has garnered significant attention in recent years. Despite the impressive performance, existing methods heavily rely on the large-scale video frames from both source and target domains, which is impractical under rigid resource constraints where source data is unavailable. To overcome this limitation, we propose SFDATrack, a generalized source-free domain adaptive tracker that merely leverages adverse weather samples from the target domain for robust state estimation. Specifically, SFDATrack first employs a mean-teacher backbone with Dual Interactive Mamba (DIM) blocks to distill the candidate target tokens that are resilient to weather variations from classified, augmented samples. Afterwards, we introduce a hyperspherical prototype projection (HPP) module to project these tokens onto multi-domain prototypes within a latent hyperspherical space. By enforcing both domain-specific and domain-invariant properties of the multi-domain prototypes, SFDATrack can be seamlessly adapted to diverse weather conditions with powerful generalizability. Extensive experiments evaluated on various benchmarks demonstrate that SFDATrack achieves superior performance compared to state-of-the-art approaches. The code is available at https://github.com/watcherBR0/sfdatrack.

## 1. 核心问题与整体含义（研究动机和背景）
- **问题**：现有域自适应视觉跟踪方法（DAVOT）在恶劣天气（如夜晚、雾、雨）下表现显著下降，且它们通常依赖源域和目标域的大规模视频帧进行训练。但在实际场景中，由于存储限制、计算成本或数据隐私，源域数据往往不可用。
- **背景**：先前工作如UMDATrack等需同时访问源数据和目标数据，而本文首次研究了**无源域自适应跟踪（Source-Free Domain Adaptive Tracking）**，仅使用目标域中的无标注恶劣天气样本进行模型适应。
- **整体目标**：构建一个无需原始源数据、能泛化到多种未知天气条件的鲁棒跟踪器。

## 2. 方法论：核心思想、关键技术细节与算法流程
### 2.1 整体框架
- 采用**均值教师（Mean-Teacher）** 结构：教师网络基于弱增强视频帧生成伪标签，学生网络接受强增强版本，两者共享架构，教师参数通过指数移动平均（EMA）更新。
- 包含两个核心模块：**双交互曼巴（Dual Interactive Mamba, DIM）** 和**超球原型投影（Hyperspherical Prototype Projection, HPP）**。

### 2.2 双交互曼巴模块（DIM）
- **目的**：通过双向序列交互和上下文标记交换，缩小不同增强视图间的领域差异，提取对天气变化鲁棒的候选目标标记。
- **流程**：
  1. 将教师和学生网络的搜索令牌（search tokens）翻转（reverse），并与可学习的交互提示令牌（interactive prompt tokens）拼接，实现跨分支信息交换。
  2. 通过线性投影、sigmoid门控，生成门控向量，然后输入共享的曼巴状态空间模型（SSM）。
  3. 融合两个分支的门控特征，得到增强后的搜索令牌。
- **关键**：利用状态空间模型（Mamba）替代Transformer，避免二次复杂度，并通过双向设计增强跨视图对齐。

### 2.3 超球原型投影模块（HPP）
- **目的**：将DIM输出的特征投影到潜在超球空间中的多域原型上，同时强制执行**域特定对齐（Domain-Specific Alignment, DSA）** 和**域不变对齐（Domain-Invariant Alignment, DIA）**，提升泛化能力。
- **域特定对齐（DSA）**：对每个天气域，构建学生网络的相似性矩阵与教师网络的软分配目标之间的交叉熵损失，确保相同天气条件的学生预测与教师一致。
- **域不变对齐（DIA）**：通过k-means聚类得到域簇，融合簇加权特征得到域不变表示，再投影到原型空间，用KL散度对齐教师和学生的分布，消除域偏差。
- **软分配**：使用Sinkhorn-Knopp算法求解最优传输问题，获得平滑的分配矩阵Q。

### 2.4 总损失函数
- 目标监督损失 \(L_S\)（分类、L1、GIoU） + 域特定对齐损失 \(L_{DSA}\) + 域不变对齐损失 \(L_{DIA}\)，权重分别为 \(\omega\) 和 \(\lambda\)（实验中均设为0.5）。
- 训练时仅使用目标域无标注数据，源域数据完全不可见。
