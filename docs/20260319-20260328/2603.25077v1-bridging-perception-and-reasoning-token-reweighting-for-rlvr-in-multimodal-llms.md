## 摘要
将具有可验证奖励的强化学习（RLVR）扩展到多模态大语言模型（MLLMs）面临一个根本性挑战：它们的响应本质上交织着感知相关 token（用于定位视觉内容）和推理相关 token（用于构建推理链）。这些 token 类型实例化了不同但又相互依赖的能力——视觉定位和符号推理——这使得孤立的优化变得不足。通过 token 级别的实证分析，我们证明了仅优化感知或仅优化推理 token 的表现始终不如全面优化，这强调了它们内在的耦合性。为了解决这个问题，我们提出了一种即插即用的 Token 重加权（ToR）策略，该策略通过识别这两类关键 token 并在 RLVR 训练期间动态重新加权，显式地对这种相互依赖性进行建模。在现有方法（如 GRPO 和 DAPO）的基础上应用 ToR，在多个多模态推理基准测试中实现了持续的性能提升，在准确的视觉定位和连贯的推理方面均达到了最先进的性能。

## Abstract
Extending Reinforcement Learning with Verifiable Rewards (RLVR) to multimodal large language models (MLLMs) faces a fundamental challenge: their responses inherently interleave perception-related tokens, which ground visual content, with reasoning-related tokens, which construct reasoning chains. These token types instantiate distinct yet interdependent capacities -- visual grounding and symbolic reasoning -- making isolated optimization insufficient. Through token-level empirical analysis, we demonstrate that optimizing either perception- or reasoning-only tokens consistently underperforms full optimization, underscoring their inherent coupling. To address this, we propose a plug-and-play Token-Reweighting (ToR) strategy that explicitly models this interdependence by identifying critical tokens of both types and dynamically reweighting them during RLVR training. Applied on top of existing methods (e.g., GRPO and DAPO), ToR delivers consistent performance gains across multiple multi-modal reasoning benchmarks, achieving state-of-the-art performance with both accurate visual grounding and coherent reasoning.

### 1. 核心问题与研究动机
*   **核心问题**：如何有效提升多模态大语言模型（MLLM）在复杂任务中的推理能力？
*   **研究背景**：现有的可验证奖励强化学习（RLVR）在纯文本领域（如 DeepSeek-R1）表现优异，但在多模态场景下，模型的回复交织着**感知 Token**（用于锚定视觉内容）和**推理 Token**（用于逻辑推导）。
*   **研究动机**：作者发现感知与推理在 Token 层面是高度耦合的。如果只优化推理 Token，模型会产生逻辑通顺但视觉事实错误的“幻觉”；如果只优化感知 Token，模型则无法构建连贯的逻辑链。因此，需要一种能够同时兼顾并平衡这两者权重的优化策略。

### 2. 方法论
*   **核心思想**：提出 **Token-Reweighting (ToR)** 策略。这是一种即插即用的模块，通过识别回复中关键的感知和推理 Token，并在强化学习的梯度计算中动态调整它们的权重。
*   **关键技术细节**：
    *   **推理相关 Token 识别**：利用**预测熵（Entropy）**。高熵位置通常代表模型在逻辑推导时的关键决策点（Forking points）。
    *   **感知相关 Token 识别**：利用**视觉敏感度（Visual Sensitivity）**。通过对比“有图像输入”和“无图像输入（空占位符）”时 Token 的对数概率差异（logp-diff）来定位。
    *   **重加权机制**：在 GRPO 或 DAPO 的目标函数中，为属于推理集合 $T_r$ 和感知集合 $T_p$ 的 Token 分别分配权重系数 $\gamma_r$ 和 $\gamma_p$。
*   **算法流程**：
    1.  模型生成多个 Rollout 响应。
    2.  计算每个 Token 的熵和视觉敏感度。
    3.  根据预设比例（如 Top 30%）筛选出关键 Token 集合。
    4.  在计算策略梯度损失时，仅对这些关键 Token 应用加权后的优势函数（Advantage），其余 Token 权重设为零或较低值。

*   **数据集**：
    *   **训练集**：Geometry3K (2.1K 样本) 和 ViRL-39K。
    *   **评测基准**：视觉推理（MathVerse, MathVision, MathVista, WeMath）和视觉感知（HalluBench）。
*   **对比方法**：
    *   **开源指令微调模型**：InternVL-2.5/3, LLaVA-OneVision, Qwen2.5-VL-7B-Instruct。
    *   **RLVR 基准方法**：R1-VL, Vision-R1, MM-Eureka, NoisyRollout, 以及原始的 GRPO 和 DAPO。
*   **模型骨干**：主要基于 Qwen2.5-VL-7B 和 3B 版本。

*   **训练配置**：学习率 1e-6，全局 Batch Size 为 128，Rollout Batch Size 为 512，每个样本生成 12 个响应（Rollout n=12）。
*   **框架**：基于 EasyR1 强化学习训练框架。

### 5. 实验数量与充分性
*   **实验规模**：作者进行了多维度的实验，包括在不同规模数据集（2.1K vs 39K）和不同参数量模型（3B vs 7B）上的验证。
*   **消融实验**：非常详尽。测试了不同 Token 选取比例（20% 到 80%）、不同权重组合（$\gamma_p$ 从 0.1 到 1.5）、以及多种感知 Token 识别指标（概率差、对数概率差、熵差等）的对比。
*   **充分性评价**：实验设计客观且公平，通过控制变量法清晰地展示了“孤立优化”与“联合优化”的性能差距，证明了 ToR 的稳健性。

### 6. 主要结论与发现
*   **耦合性发现**：实验证明，仅优化推理或仅优化感知 Token 的效果均差于全量优化，而 ToR 的联合重加权效果最优。
*   **性能提升**：ToR-DAPO 在 Geometry3K 训练集上，将 MathVerse 的准确率从 50.6% 提升至 53.4%，WeMath 从 69.3% 提升至 72.1%。
*   **泛化能力**：该策略在不同模型规模和数据规模下均表现出一致的增益，证明了其作为通用 RLVR 增强工具的潜力。

### 7. 优点
*   **即插即用**：不需要修改模型结构或引入额外的奖励模型，计算开销增加极小。
*   **自监督信号**：利用模型自身的预测熵和概率差作为信号，无需昂贵的人工标注或外部模型辅助。
*   **理论与实证结合**：通过对 Token 分布的深入分析（如熵分布图），为重加权策略提供了坚实的实证支持。

### 8. 不足与局限
*   **感知识别开销**：为了计算视觉敏感度，Rollout 阶段需要进行两次前向传播（有图和无图），这会增加约一倍的推理耗时。
*   **Token 识别精度**：目前的识别方式仍属于粗粒度的统计代理，可能无法完全捕捉到极其细微的视觉-语义关联。
*   **应用范围**：实验主要集中在数学和几何推理任务，对于更通用的多模态对话或视频理解任务的有效性尚需进一步验证。
