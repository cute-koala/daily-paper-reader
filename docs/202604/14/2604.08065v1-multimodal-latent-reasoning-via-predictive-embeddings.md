## 摘要
工具增强的多模态推理使视觉语言模型能够通过与外部工具（例如，裁剪、深度估计）交互来提升感知能力。然而，此类方法会产生大量推理开销，需要专门的监督，并且容易产生错误的工具调用。我们提出了Pearl（潜在空间推理的预测性嵌入对齐），这是一个受JEPA启发的框架，完全在潜在空间中从专家工具使用轨迹中学习，从而在推理时无需显式调用工具。与基于重建的潜在推理方法不同——后者自回归地生成潜在标记并遭受训练-推理不匹配以及对多步骤工具使用的有限支持——Pearl直接从多模态轨迹中学习预测性嵌入，同时保留了标准的视觉语言生成流程：它不依赖于特定模型、易于训练，并且天然支持包含多次工具调用的轨迹。在多个感知基准测试上的实验表明，Pearl达到或超越了标准的有监督微调以及基于重建的潜在推理方法。此外，我们提供的经验证据表明，基于重建的方法主要学习的是嵌入而非潜在空间中的图像编辑，这促使我们将预测性嵌入学习作为一种更具原则性的替代方案。

## Abstract
Tool-augmented multimodal reasoning enables visual language models (VLMs) to improve perception by interacting with external tools (e.g., cropping, depth estimation). However, such approaches incur substantial inference overhead, require specialized supervision, and are prone to erroneous tool calls. We propose Pearl (Predictive Embedding Alignment for Reasoning in Latent space), a JEPA-inspired framework that learns from expert tool-use trajectories entirely in the latent space, eliminating the need for explicit tool invocation at inference time. Unlike reconstruction-based latent reasoning methods, which autoregressively generate latent tokens and suffer from training-inference mismatch and limited support for multi-step tool use, Pearl directly learns predictive embeddings from multimodal trajectories while preserving the standard vision-language generation pipeline: it is model-agnostic, simple to train, and naturally supports trajectories with multiple tool calls. Experiments across multiple perception benchmarks show that Pearl matches or outperforms standard supervised fine-tuning and reconstruction-based latent reasoning approaches. Furthermore, we provide empirical evidence that reconstruction-based methods primarily learn embeddings rather than image edits in latent space, motivating predictive embedding learning as a more principled alternative.

## 1. 核心问题与整体含义

    2.  **需要专门监督**：学习何时、如何正确调用工具需要精心设计的专家轨迹数据进行监督训练。
    3.  **易出错且脆弱**：错误的工具调用会浪费计算资源并污染上下文信息。
*   **研究背景**：为解决上述问题，近期工作探索了“潜在推理”，即让模型在连续的嵌入空间中操作，而非生成离散的中间输出。其中，基于重构的潜在推理方法通过自回归生成潜在令牌来“想象”中间图像编辑，但存在**训练-推理不匹配**（训练时使用大量令牌，推理时只解码少量）和**难以支持多步工具调用**的局限。
*   **整体含义**：本文提出PEARL框架，其核心思想是摒弃复杂的自回归潜在令牌生成与重构，转而直接学习从原始输入到整个专家轨迹的预测性嵌入。这旨在将工具使用的效果完全内化到模型的表征空间中，从而在保持标准、高效推理流程的同时，获得与显式工具调用相当甚至更优的性能。

## 2. 提出的方法论
*   **核心思想**：受联合嵌入预测架构（JEPA）启发，将原始图像-问题对 `<I0, Q>` 和专家多模态推理轨迹 `R = (I1, T1, ..., IN, TN)` 视为同一问题的两个“视图”。模型的目标是直接从输入视图预测出轨迹视图在潜在空间中的嵌入表示，从而内化工具使用的效果和推理步骤。
*   **关键技术细节与流程**：
    1.  **双视图编码**：使用现成的VLM分别对输入 `<I0, Q>` 和完整轨迹 `R` 进行独立前向传播，取最后一个令牌的最终隐藏状态作为各自的序列表示 `h_x` 和 `h_R`。
    2.  **绑定权重的预测器**：在输入序列后附加K个可学习的 `[PRED]` 特殊令牌。利用VLM自身的注意力层对这些令牌进行计算，将最后一个 `[PRED]` 令牌的隐藏状态作为预测的轨迹嵌入 `\hat{h}_R`。这复用模型参数，无需引入额外网络。
    3.  **三重损失函数联合优化**：
        *   **自回归生成损失 (L_VLM)**：标准的下一个词预测损失，确保模型保留文本生成能力以输出最终答案。
        *   **JEPA预测损失 (L_JEPA)**：使用 SmoothL1 Loss 对齐预测嵌入 `\hat{h}_R` 与停止梯度的目标嵌入 `sg[h_R]`。这是核心目标，驱使模型学习从输入到轨迹效果的紧凑表征。
        *   **下一潜在状态预测损失 (L_NextLat)**：鼓励每个时间步的隐藏状态能够预测未来若干步的状态，使其收敛为能概括过去历史的“信念状态”，从而提升作为编码目标的序列表示质量。
    4.  **总损失**: `L_PEARL = L_VLM + λ [L_JEPA + L_NextLat]`
*   **推理流程**：极其简单。仅需输入原始图像和问题 `<I0, Q>` ，像标准VLM一样直接生成文本答案。无需调用任何外部工具、无需解码中间图像或潜在令牌、也无需使用 `[PRED]` 令牌。

## 9. 潜在影响与未来方向
