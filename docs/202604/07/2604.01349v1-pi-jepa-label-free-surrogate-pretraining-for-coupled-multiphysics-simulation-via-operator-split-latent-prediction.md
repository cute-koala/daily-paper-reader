## 摘要
油藏模拟工作流程面临一个根本性的数据不对称问题：输入参数场（地质统计渗透率实现、孔隙度分布）可以任意大量生成，但现有的神经算子代理模型需要大量昂贵的标记模拟轨迹数据，无法利用这些未标记的结构。我们提出了\textbf{PI-JEPA}（物理信息联合嵌入预测架构），这是一种代理预训练框架，它\emph{无需任何完整的偏微分方程求解}即可进行训练，方法是在每个子算子偏微分方程残差正则化下对未标记参数场进行掩码潜在预测。预测器组的结构与主导方程的Lie--Trotter算子分裂分解对齐，为每个子过程（压力、饱和度输运、反应）分配一个独立的物理约束潜在模块，从而使得仅用少至100个标记模拟运行即可进行微调。在单相达西流问题上，当$N_\ell{=}100$时，PI-JEPA的误差比FNO低$1.9$倍，比DeepONet低$2.4$倍；当$N_\ell{=}500$时，相比纯监督训练有24%的提升。这表明无标签代理预训练显著降低了部署多物理场代理模型所需的模拟计算成本。

## Abstract
Reservoir simulation workflows face a fundamental data asymmetry: input parameter fields (geostatistical permeability realizations, porosity distributions) are free to generate in arbitrary quantities, yet existing neural operator surrogates require large corpora of expensive labeled simulation trajectories and cannot exploit this unlabeled structure. We introduce \textbf{PI-JEPA} (Physics-Informed Joint Embedding Predictive Architecture), a surrogate pretraining framework that trains \emph{without any completed PDE solves}, using masked latent prediction on unlabeled parameter fields under per-sub-operator PDE residual regularization. The predictor bank is structurally aligned with the Lie--Trotter operator-splitting decomposition of the governing equations, dedicating a separate physics-constrained latent module to each sub-process (pressure, saturation transport, reaction), enabling fine-tuning with as few as 100 labeled simulation runs. On single-phase Darcy flow, PI-JEPA achieves $1.9\times$ lower error than FNO and $2.4\times$ lower error than DeepONet at $N_\ell{=}100$, with 24\% improvement over supervised-only training at $N_\ell{=}500$, demonstrating that label-free surrogate pretraining substantially reduces the simulation budget required for multiphysics surrogate deployment.

### **PI-JEPA 论文深度分析总结**

#### **1. 核心问题与整体含义**
*   **研究动机**：针对油藏模拟等科学计算领域存在的**数据不对称性**问题。具体表现为：输入参数场（如渗透率）可以基于地质统计模型廉价、大量生成，但获取其对应的输出解场（如压力、饱和度）则需要运行昂贵的高保真数值模拟器。现有神经算子代理模型（如FNO, DeepONet）依赖大量标记的“输入-输出”对进行监督训练，无法利用海量的无标签参数场数据。
*   **整体含义**：本文提出了一种名为 **PI-JEPA** 的新框架，旨在通过**自监督预训练**来显著降低代理模型对昂贵标记数据的依赖。其核心思想是模仿人类“从观察中学习物理规律”的过程，让模型仅从无标签的参数场和解的快照中学习物理系统的潜在结构和动力学，然后仅需少量标记数据进行微调即可获得高性能代理模型。

#### **2. 方法论**
*   **核心思想**：
    1.  **联合嵌入预测架构 (JEPA)**：借鉴计算机视觉中的I-JEPA思想，通过预测输入数据中被掩码部分的潜在表示（而非像素值）来学习有意义的表征。
    2.  **物理信息正则化 (Physics-Informed)**：在自监督预测损失的基础上，引入基于控制偏微分方程(PDE)的残差损失作为正则项，确保学习到的表征符合物理规律。
    3.  **算子分裂对齐 (Operator-Split Alignment)**：关键创新。将JEPA中的多个预测器与数值求解器中使用的Lie-Trotter算子分裂步骤一一对应（例如，压力求解对应一个预测器，饱和度输运对应另一个）。这使得每个预测器可以专门学习特定物理子过程（具有不同时间尺度）的动态。
*   **关键技术细节**：
    *   **架构组件**：
        *   **上下文编码器 (fθ)**：处理可见（上下文）区域的数据块。
        *   **目标编码器 (fξ)**：处理完整数据以生成稳定的预测目标，其权重是上下文编码器的指数移动平均(EMA)。
        *   **预测器组 ({gϕk})**：K个轻量级Transformer模块，每个对应一个物理子算子。它们以链式方式工作，将上下文潜在表示逐步“推进”到目标状态。
        *   **辅助解码器 ({dψk})**：仅用于训练阶段，将每个预测器的输出映射回物理空间以计算PDE残差损失。
    *   **训练目标函数**: `L = L_pred + λp Σ L_phys^(k) + λr L_reg`
        *   `L_pred`: 在潜在空间比较最终预测与目标编码器输出的均方误差。
        *   `L_phys^(k)`: 每个子算子的PDE残差损失（通过辅助解码器计算）。
        *   `L_reg`: VICReg正则化项，防止潜在表示崩溃。
    *   **掩码策略**: 采用时空块掩码——上下文块取自时间t的一个连续空间区域，目标块取自时间t+Δt的一个空间上位移的区域。这迫使模型学习连接两个区域的因果动力学（如对流、扩散）。

#### **6. 主要结论**
*   **核心发现**：
    1.  **显著的数据效率提升**：在单相达西流任务中，PI-JEPA预训练模型仅需约50个标记样本即可达到FNO使用500个标记样本的性能水平，实现了**约10倍的样本效率提升**。这直接解决了论文提出的核心问题。
    2.  **架构与预训练的协同效应**：消融实验表明，“Scratch”版本的PI-JEPA（无预训练）性能优于或持平FNO，证明了其架构设计的有效性。而经过自监督预训练后，性能得到进一步显著提升，证明了预训练策略的成功。

    4.  **算子分裂对齐的有效性**：通过将预测器与物理子过程对齐，模型能够更高效地捕捉多尺度、多物理场的耦合动力学。这在复杂的多相流和反应-扩散系统中尤为重要。
