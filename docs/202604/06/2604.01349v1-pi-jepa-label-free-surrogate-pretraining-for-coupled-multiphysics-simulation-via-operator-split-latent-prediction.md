## 摘要
油藏模拟工作流程面临一个根本性的数据不对称问题：输入参数场（地质统计渗透率实现、孔隙度分布）可以任意大量生成，然而现有的神经算子代理模型需要大量昂贵的标记模拟轨迹数据，无法利用这些未标记的结构。我们提出了\textbf{PI-JEPA}（物理信息联合嵌入预测架构），这是一种代理预训练框架，它\emph{无需任何完整的偏微分方程求解}即可进行训练，方法是在每个子算子偏微分方程残差正则化下对未标记参数场进行掩码潜在预测。预测器组的结构与主导方程的Lie--Trotter算子分裂分解对齐，为每个子过程（压力、饱和度输运、反应）分配一个独立的物理约束潜在模块，从而能够仅用少至100个标记模拟运行进行微调。在单相达西流问题上，当$N_\ell{=}100$时，PI-JEPA的误差比FNO低$1.9\times$，比DeepONet低$2.4\times$；当$N_\ell{=}500$时，相比纯监督训练有24%的提升。这表明无标签代理预训练显著降低了部署多物理场代理模型所需的模拟计算量。

## Abstract
Reservoir simulation workflows face a fundamental data asymmetry: input parameter fields (geostatistical permeability realizations, porosity distributions) are free to generate in arbitrary quantities, yet existing neural operator surrogates require large corpora of expensive labeled simulation trajectories and cannot exploit this unlabeled structure. We introduce \textbf{PI-JEPA} (Physics-Informed Joint Embedding Predictive Architecture), a surrogate pretraining framework that trains \emph{without any completed PDE solves}, using masked latent prediction on unlabeled parameter fields under per-sub-operator PDE residual regularization. The predictor bank is structurally aligned with the Lie--Trotter operator-splitting decomposition of the governing equations, dedicating a separate physics-constrained latent module to each sub-process (pressure, saturation transport, reaction), enabling fine-tuning with as few as 100 labeled simulation runs. On single-phase Darcy flow, PI-JEPA achieves $1.9\times$ lower error than FNO and $2.4\times$ lower error than DeepONet at $N_\ell{=}100$, with 24\% improvement over supervised-only training at $N_\ell{=}500$, demonstrating that label-free surrogate pretraining substantially reduces the simulation budget required for multiphysics surrogate deployment.

## 2. 提出的方法论
*   **核心思想**：将自监督学习中的联合嵌入预测架构与物理信息约束相结合，并使其结构与物理方程的数值求解过程对齐。
    *   **无标签预训练**：模仿JEPA/I-JEPA的思想，在潜在空间进行预测任务。将解场划分为上下文块和目标块，让模型根据上下文块的潜在编码去预测未来时刻目标块的潜在编码，全程无需真实解场标签。
    *   **物理信息正则化**：在预测损失之外，引入基于PDE残差的物理约束损失，确保学习到的表示符合物理规律。
    *   **算子分裂对齐**：关键创新在于将预测器组的结构与求解耦合PDE的Lie-Trotter算子分裂分解对齐。每个物理子过程（如压力求解、饱和度输运、化学反应）对应一个独立的潜在预测器模块，并配备专属的PDE残差正则项。

*   **关键技术细节**：
    1.  **架构组件**：
        *   **上下文编码器 $f_\theta$**：处理上下文块，生成潜在编码。
        *   **目标编码器 $f_\xi$**：处理完整/目标块，提供稳定的预测目标。其权重 $\xi$ 是 $\theta$ 的指数移动平均。
        *   **预测器组 $\{g_{\phi_k}\}$**：一系列轻量级Transformer模块，每个对应一个物理子算子。它们以链式方式工作，逐步更新潜在状态。
        *   **辅助解码器 $d_{\psi_k}$**：仅用于训练阶段，将每个预测器的输出映射回物理空间以计算PDE残差。
    2.  **训练目标函数**：
        $$ \mathcal{L}(\theta, \{\phi_k\}) = \mathcal{L}_{pred} + \lambda_p \sum_{k=1}^{K} \mathcal{L}^{(k)}_{phys} + \lambda_r \mathcal{L}_{reg} $$
        *   $\mathcal{L}_{pred}$: 潜在空间预测损失（L2误差）。
        *   $\mathcal{L}^{(k)}_{phys}$: 第k个子算子的PDE残差损失。
        *   $\mathcal{L}_{reg}$: VICReg风格的协方差正则化项，防止潜在表示坍塌。
    3.  **掩码策略**：采用时空块掩码。上下文块取自当前时刻的一个连续空间区域，目标块取自下一时刻的一个空间上偏移的连续区域。这迫使模型学习连接上下文与目标的因果动力学（对流、扩散等）。
    4.  **推理流程**：微调后，通过自回归方式在潜在空间进行多步推演。使用预训练的编码器和微调后的预测头进行闭环生成。
