## 摘要
本文针对部分可观测系统中的强化学习策略训练问题，提出了一种利用仅在训练阶段可用的特权、任意时间可行规划器的方法。我们将此形式化为一个部分可观测马尔可夫决策过程，其中，一个拥有近似动力学模型和特权状态信息的规划器代理，指导一个仅能观测到真实状态有损投影的学习代理。为实现该框架，我们引入了一种作为规划器代理的任意时间可行模型预测控制算法。对于学习代理，我们提出了规划器到策略的软演员-评论家方法，该方法通过提炼规划器代理的特权知识来缓解部分可观测性，从而提高样本效率和最终策略性能。我们通过严格的理论分析支持该框架。最后，我们在NVIDIA Isaac Lab仿真环境中验证了所提方法，并成功将其部署在现实世界的Unitree Go2四足机器人上，使其能够在复杂、障碍物丰富的环境中导航。

## Abstract
This paper addresses the problem of training a reinforcement learning (RL) policy under partial observability by exploiting a privileged, anytime-feasible planner agent available exclusively during training. We formalize this as a Partially Observable Markov Decision Process (POMDP) in which a planner agent with access to an approximate dynamical model and privileged state information guides a learning agent that observes only a lossy projection of the true state. To realize this framework, we introduce an anytime-feasible Model Predictive Control (MPC) algorithm that serves as the planner agent. For the learning agent, we propose Planner-to-Policy Soft Actor-Critic (P2P-SAC), a method that distills the planner agent's privileged knowledge to mitigate partial observability and thereby improve both sample efficiency and final policy performance. We support this framework with rigorous theoretical analysis. Finally, we validate our approach in simulation using NVIDIA Isaac Lab and successfully deploy it on a real-world Unitree Go2 quadruped navigating complex, obstacle-rich environments.

### **论文总结：PriPG-RL**

#### **1. 核心问题与整体含义**
*   **研究动机**：在部分可观测马尔可夫决策过程（POMDP）中，标准强化学习方法（如SAC、PPO）因状态混淆和值函数不稳定而面临训练困难、样本效率低下和性能不佳的问题。现有方法（如基于演示的RL或启发式算法正则化）通常假设完全可观测性，难以有效解决POMDP中的根本挑战。
*   **核心思想**：提出一个名为**Privileged Planner-Guided RL (PriPG-RL)** 的通用框架。该框架的核心是利用一个仅在训练时可用的**特权规划器代理**来指导学习代理。规划器拥有近似动力学模型和完整状态信息（特权信息），而学习代理仅能观测到有损的状态投影。通过将规划器的知识“蒸馏”给学习代理，旨在缓解部分可观测性带来的负面影响。

#### **2. 方法论**
*   **整体框架**：
    *   **规划器代理 (Planner Agent)**：采用一种基于REAP（Robust-to-Early Termination MPC）的**任意时间可行MPC算法**。该算法通过一个原始-对偶梯度流动态系统求解优化问题，保证在任何计算时刻都能输出可行（但不一定最优）的控制动作，非常适合作为在线训练中的稳定指导信号。
    *   **学习代理 (Learning Agent)**：提出**Planner-to-Policy Soft Actor-Critic (P2P-SAC)** 算法，在标准SAC基础上融合了四个关键机制：
        1.  **双经验回放池 (Dual Replay Buffer)**：包含一个存储规划器轨迹的固定缓冲池和一个存储在线交互数据的FIFO缓冲池。
        2.  **确定性三阶段成熟度调度 (Deterministic Three-Phase Maturity Schedule)**：定义了“平台期-退火期-成熟期”，控制何时使用规划器动作进行行为替代以及何时激活优势门控。
        3.  **Logit空间模仿锚点 (Logit-Space Imitation Anchor)**：将规划器的动作映射到策略网络的logit空间进行正则化，避免了在动作边界处梯度消失的问题。
        4.  **基于优势值的Sigmoid门控 (Advantage-Based Sigmoid Gate)**：在成熟期，根据学习策略相对于规划器的估计优势值动态调整模仿损失的权重。当学习策略更优时抑制模仿，当规划器更优时保留指导。
*   **理论贡献**：
    *   证明了REAP满足“任意时间可行规划器”的定义。
    *   推导了P2P-SAC的目标函数梯度形式（定理1），表明其正则项将策略拉向门控加权平均后的规划器logits，并且与状态混淆引起的不可约方差无关。
