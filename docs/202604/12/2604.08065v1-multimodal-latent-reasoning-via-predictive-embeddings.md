## 摘要
工具增强的多模态推理使视觉语言模型（VLMs）能够通过与外部工具（例如，裁剪、深度估计）交互来改进感知。然而，此类方法会产生大量的推理开销，需要专门的监督，并且容易出现错误的工具调用。我们提出了 Pearl（潜在空间推理的预测嵌入对齐），这是一个受 JEPA 启发的框架，完全在潜在空间中从专家工具使用轨迹中学习，从而在推理时无需显式调用工具。与基于重建的潜在推理方法不同——后者自回归地生成潜在标记并遭受训练-推理不匹配以及对多步骤工具使用的有限支持——Pearl 直接从多模态轨迹中学习预测嵌入，同时保留了标准的视觉语言生成流程：它是模型无关的、易于训练的，并且天然支持包含多个工具调用的轨迹。在多个感知基准测试上的实验表明，Pearl 达到或超越了标准监督微调和基于重建的潜在推理方法的性能。此外，我们提供的经验证据表明，基于重建的方法主要学习的是嵌入而非潜在空间中的图像编辑，这促使我们将预测嵌入学习作为一种更具原则性的替代方案。

## Abstract
Tool-augmented multimodal reasoning enables visual language models (VLMs) to improve perception by interacting with external tools (e.g., cropping, depth estimation). However, such approaches incur substantial inference overhead, require specialized supervision, and are prone to erroneous tool calls. We propose Pearl (Predictive Embedding Alignment for Reasoning in Latent space), a JEPA-inspired framework that learns from expert tool-use trajectories entirely in the latent space, eliminating the need for explicit tool invocation at inference time. Unlike reconstruction-based latent reasoning methods, which autoregressively generate latent tokens and suffer from training-inference mismatch and limited support for multi-step tool use, Pearl directly learns predictive embeddings from multimodal trajectories while preserving the standard vision-language generation pipeline: it is model-agnostic, simple to train, and naturally supports trajectories with multiple tool calls. Experiments across multiple perception benchmarks show that Pearl matches or outperforms standard supervised fine-tuning and reconstruction-based latent reasoning approaches. Furthermore, we provide empirical evidence that reconstruction-based methods primarily learn embeddings rather than image edits in latent space, motivating predictive embedding learning as a more principled alternative.

    3.  **鲁棒性差**：错误的工具调用会浪费计算资源并污染上下文信息。
    4.  **扩展性有限**：现有方法通常局限于单步或同质化工具操作，难以支持复杂的多步骤、多类型工具链。

*   **整体含义**：本文旨在解决上述问题，提出一种新的范式——让模型在潜在空间中“内化”工具使用的效果，从而在推理时完全摆脱对显式工具的依赖，实现高效、鲁棒且支持多步操作的多模态推理。

## 2. 方法论
*   **核心思想**：受联合嵌入预测架构（JEPA）启发，提出 PEARL（Predictive Embedding Alignment for Reasoning in Latent space）框架。其核心是学习一个从原始图像-问题对到专家多模态推理轨迹的预测性嵌入映射，而非重建中间图像。
*   **关键技术细节**：
    *   **输入与轨迹视图**：
        *   输入视图 `x = ⟨I0, Q⟩` ：原始图像和问题。
        *   轨迹视图 `R = (I1, T1, ..., IN, TN)` ：专家使用工具生成的中间图像序列与对应的推理文本交错序列。
    *   **独立编码与预测器**：使用同一个预训练 VLM（如 Qwen-VL）对 `x` 和 `R` 进行两个独立的前向传播，得到各自的最终隐藏状态表示 `h_x` 和 `h_R`。然后通过添加 K 个可学习的 **[PRED]** token作为“预测器”，将 `h_x` 映射到预测的轨迹嵌入 `ĥ_R`。
    *   **训练目标函数 (`L_PEARL`)**: 
        1.  **JEPA对齐损失 (`L_JEPA`)**: Smooth L1 Loss，用于对齐预测嵌入 `ĥ_R` 和停止梯度的目标嵌入 `sg[h_R]`。这是核心的预测学习信号。
        2.  **自回归生成损失 (`L_VLM`)**: 标准的交叉熵损失，确保模型保留生成最终答案文本的能力。
        3.  **下一潜在状态预测损失 (`L_NextLat`)**: Smooth L1 Loss，鼓励模型的隐藏状态成为能够预测未来状态的“信念状态”，从而提升作为编码目标的表征质量。这是一个正则化项。
        *   总损失为三者加权和: `L_PEARL = L_VLM + λ [L_JEPA + L_NextLat]`
    *   **推理流程**: PEARL的推理极其简单——仅需输入原始图像-问题对 `<I0, Q>` ，然后像标准VLM一样直接生成答案文本。无需调用任何外部工具、无需解码潜在token、也无需使用 **[PRED]** token。

## 10. 总结
本文提出的 PEARL 框架是对现有基于显式工具调用的多模态推理方法的一次重要反思与革新。它通过将联合嵌入预测架构（JEPA）引入视觉语言模型，成功地让模型在潜空间中学习并内化了专家使用工具的“效果”，从而在测试时实现了零额外开销的高效推理。

实验表明，PEARL 在多种感知密集型任务上能够匹配甚至超越监督微调和基于重建的潜在推理方法，同时具备模型无关性和对多步、多类型操作的天然支持能力。更重要的是，论文通过对竞争方法的深入分析，揭示了基于重建的方法其性能提升主要源于学习到了有用的嵌入表征，而非真正模拟视觉变换过程，从而从反面论证了 PEARL “直接学习预测嵌入”这一路径的原则性优势。

尽管存在可解释性较差、训练计算成本较高等局限，但 PEARL 为多模态推理开辟了一条高效、简洁且极具潜力的新方向。它将研究的焦点从“如何调用工具”转向了“如何内化工具的效果”，推动我们思考如何让模型在抽象的表示空间中直接进行复杂的组合式推理。
