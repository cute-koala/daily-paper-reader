## 摘要
参数高效的提示学习已成为将视觉语言模型适应下游任务的事实标准。现有方法主要侧重于将文本提示与一阶视觉特征（即空间特征图）对齐。虽然这对于细粒度语义区分是有效的，但我们认为仅依赖一阶信息不足以实现稳健的适应，因为这些空间纠缠的特征极易受到领域偏移和局部噪声的影响。在这项工作中，我们提出了基于二阶统计量的\textbf{Gram锚定提示学习}框架，用于视觉语言模型，该框架将局部语义对齐与全局结构一致性协同起来。在方法上，我们通过\textbf{Gram矩阵}引入了一个额外的二阶统计流，以增强标准的一阶空间交互。通过将提示锚定到这些二阶先验上，我们的方法使语言表示能够动态适应不同领域的统计分布偏移。大量实验证明了二阶特征的有效性，并展示了GAPL在各种基准测试中的出色性能。

## Abstract
Parameter-efficient prompt learning has become the de facto standard for adapting Vision-Language Models (VLMs) to downstream tasks. Existing approaches predominantly focus on aligning text prompts with first-order visual features (i.e., spatial feature maps). While effective for fine-grained semantic discrimination, we argue that relying solely on first-order information is insufficient for robust adaptation, as these spatially entangled features are highly susceptible to domain shifts and local noise. In this work, we propose \textbf{Gram-Anchored Prompt Learning (GAPL)} for Vision-Language Models via Second-Order Statistics, a framework that synergizes local semantic alignment with global structural consistency. Methodologically, we introduce an additional second-order statistical stream via \textbf{Gram matrices} that augments the standard first-order spatial interaction. By anchoring prompts to these second-order priors, our approach enables language representations to dynamically adapt to statistical distribution shifts across diverse domains. Extensive experiments indicate the effectiveness of the second-order features, and show compelling performances of GAPL on various benchmarks.

### **1. 核心问题与整体含义**
*   **研究背景**：参数高效的提示学习已成为适配大规模视觉语言模型（如CLIP）到下游任务的主流方法。现有动态提示学习方法（如CoCoOp）主要依赖**一阶视觉特征**（如全局池化特征或空间特征图）来生成图像条件化的文本提示。
*   **核心问题**：尽管一阶特征对语义判别有效，但它们与图像的局部外观和空间细节高度耦合。当面临**领域偏移**（例如从真实照片到素描）时，同一语义概念在不同域的特征空间中可能相距甚远，导致生成的提示对领域风格变化敏感，而非稳定的类别结构。这限制了模型在跨域场景下的鲁棒性。
*   **整体含义**：本文提出，除了语义信息外，模型的适应过程还需要一个更稳定、能捕捉图像级统计特性的信号。受风格迁移和领域自适应研究的启发，作者认为源自视觉特征的**Gram矩阵所蕴含的二阶统计量**能提供这种全局纹理和外观信息，可作为增强提示学习鲁棒性的“锚点”。

### **2. 方法论**
*   **核心思想**：提出 **Gram-Anchored Prompt Learning (GAPL)** 框架。其核心是在传统的基于一阶特征的动态提示学习基础上，引入一个额外的、基于二阶统计量的分支（Gram-Anchored Stream），为文本表示提供一个关于图像整体结构和风格的“锚定”信号。
*   **关键技术细节与流程**：
    *   **三流架构**：GAPL包含三个并行且互补的预测流：
        1.  **全局不变流 (Global Invariant Stream)**：沿用经典方法，将学习的文本提示特征与CLS令牌的全局视觉特征对齐，保留预训练模型的通用语义知识。
        2.  **Gram锚定流 (Gram-Anchored Stream, 核心创新)**：
            *   从视觉编码器输出的`N`个`d`维补丁令牌 `F` 计算通道间的Gram矩阵 `G = F^T F / N ∈ R^{d×d}`。
            *   为保持计算高效性并提取紧凑描述符，仅保留其对角线元素 `g_diag = diag(G) ∈ R^d`（代表各通道的能量）。
            *   通过一个轻量级MLP将 `g_diag` 转换为门控向量 `Γ`。
            *   使用该门控向量对文本提示特征进行元素级调制：`a_style = w + w ⊙ Γ`，生成“风格文本锚点”。
        3.  **上下文锚定流 (Contextual-Anchored Stream)**：利用可学习的局部信号与文本提示特征进行注意力交互，生成多个上下文文本锚点，用于与局部视觉特征的细粒度对齐。
    *   **训练与推理融合**：
        *   每个流都产生独立的分类损失进行监督。
        *   采用输入自适应的分支融合策略：根据归一化的全局视觉特征和风格门控向量构成的描述符，通过一个小型MLP动态预测三个流的权重。
        *   最终预测是三个流的加权融合结果。
