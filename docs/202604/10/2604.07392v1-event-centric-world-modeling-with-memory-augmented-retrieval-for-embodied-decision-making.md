## 摘要
在动态和安全关键环境中运行的自主智能体需要兼具计算效率和物理基础的决策框架。然而，许多现有方法依赖于端到端学习，这通常缺乏可解释性以及确保与物理约束一致性的显式机制。在本工作中，我们提出了一种用于具身决策的、具有记忆增强检索功能的事件中心世界建模框架。该框架将环境表示为一组结构化的语义事件，这些事件被编码为排列不变的潜在表示。决策是通过对先前经验知识库的检索来执行的，其中每个条目将一个事件表示与相应的操作关联起来。最终动作计算为检索到的解决方案的加权组合，从而在决策与存储的经验之间建立了透明的联系。所提出的设计实现了动态环境的结构化抽象，并通过基于案例的推理支持可解释的决策。此外，将物理信息知识纳入检索过程鼓励选择与观测到的系统动力学相一致的操作。在无人机飞行场景中的实验评估表明，该框架能够在实时控制约束下运行，同时保持行为可解释且一致。

## Abstract
Autonomous agents operating in dynamic and safety-critical environments require decision-making frameworks that are both computationally efficient and physically grounded. However, many existing approaches rely on end-to-end learning, which often lacks interpretability and explicit mechanisms for ensuring consistency with physical constraints. In this work, we propose an event-centric world modeling framework with memory-augmented retrieval for embodied decision-making. The framework represents the environment as a structured set of semantic events, which are encoded into a permutation-invariant latent representation. Decision-making is performed via retrieval over a knowledge bank of prior experiences, where each entry associates an event representation with a corresponding maneuver. The final action is computed as a weighted combination of retrieved solutions, providing a transparent link between decision and stored experiences. The proposed design enables structured abstraction of dynamic environments and supports interpretable decision-making through case-based reasoning. In addition, incorporating physics-informed knowledge into the retrieval process encourages the selection of maneuvers that are consistent with observed system dynamics. Experimental evaluation in UAV flight scenarios demonstrates that the framework operates within real-time control constraints while maintaining interpretable and consistent behavior.

### 1. 核心问题与整体含义
*   **研究动机**：在动态、安全关键（如无人机飞行）环境中，自主智能体需要高效且可靠的决策框架。当前主流的端到端学习方法（如视觉-语言-动作模型）虽然有效，但普遍存在两大缺陷：
    1.  **缺乏可解释性**：决策过程是“黑箱”，难以分析和验证。
    2.  **难以保证物理一致性**：生成的行动可能违反物理约束（如动力学限制），这在安全关键应用中风险极高。
*   **整体目标**：提出一个兼具**可解释性**、**物理一致性**和**实时性**的具身决策框架。其核心思想是摒弃端到端映射，转而采用一种基于案例推理和结构化世界模型的透明化方法。

### 2. 提出的方法论
*   **核心思想 (ERA框架)**：将环境抽象为结构化的“事件”，通过检索存储在“知识库”中的先验经验（事件-动作对）来生成当前决策，并对检索到的多个方案进行加权融合。
*   **关键技术细节与流程**：
    1.  **事件中心表示 (Event-Centric Representation)**：
        *   将原始感知输入（如传感器数据）转换为一个结构化的事件列表 `Et`。每个事件元素包含对象ID、位置、运动状态、环境状态、自身状态和目标上下文等信息。
        *   使用排列不变编码器（如DeepSets或Transformer）将事件列表压缩为一个紧凑的潜在事件编码 `zt = f(Et)`。这确保了表示不受物体检测顺序影响。
    2.  **记忆增强检索 (Memory-Augmented Retrieval)**：
        *   维护一个知识库 `M`，其中存储了大量由专家演示或仿真生成的事件编码-动作对 `(zi, ai)`。
        *   对于当前事件编码 `zt`，在知识库中执行近似最近邻搜索，找到最相似的k个历史经验。
    3.  **可解释决策生成 (Interpretable Decision-Making)**：
        *   根据相似度为每个检索到的动作分配权重 `wi`（使用softmax归一化）。
        *   最终动作为加权组合：`at = Σ wi * ai`。这种机制直接建立了决策与历史案例的透明联系。
    4.  **物理信息约束 (Physics-Informed Constraints)**：
        *   **稳定性约束**：在潜在空间中引入线性动力学模型 `zt+1 = Ψ zt + Γ at + εt`，并强制要求其是收缩的（谱半径 ρ(Ψ) < 1），以保证长期预测有界且稳定。这通过Lyapunov稳定性条件实现。
        *   **聚类贝叶斯选择 (Clustered Bayesian Selection)**：为避免平均两个方向相反的安全动作导致“直行碰撞”的风险，先将检索到的动作按方向聚类，然后选择总权重最高的簇，仅在该簇内进行加权平均。这确保了动作的一致性。
*   **训练流程**：
    *   **预训练阶段 (Algorithm 1)**：使用专家数据（如虚拟势场控制器生成的轨迹）初始化知识库和编码器。损失函数包含度量学习损失和模仿学习损失。
    *   **在线训练/微调阶段 (Algorithm 2)**：在与对抗性课程交互中优化策略。损失函数结合了物理一致性正则项 `R_phys`（惩罚与检索经验在潜在空间的不一致）和任务性能目标 `J_perf`。

### 8. 缺点与局限性
1.  **知识库依赖与覆盖度瓶颈**：框架性能高度依赖于预构建的知识库 `M`。如果知识库未能覆盖某些极端或罕见情况（“长尾分布”），智能体可能无法做出有效决策。知识库的构建（收集专家数据）本身可能成本高昂。
2.  **缺乏横向性能对比**：如前所述，论文未将ERA框架与当前主流的强化学习（如PPO、SAC）或端到端视觉-语言-动作模型进行系统的定量比较。这使得评估其相对于领域内其他方法的绝对竞争力变得困难。
3.  **简化仿真环境**：实验在几何体表示的简化仿真中进行，未涉及复杂的视觉感知（如真实图像输入）、传感器噪声或更复杂的物理交互（如空气动力学扰动）。这限制了结论向真实世界机器人平台迁移的说服力。
4.  **事件表示的局限性**：事件列表的生成依赖于准确的对象检测、跟踪和状态估计模块。这些上游模块的误差会直接传播并影响决策质量。论文未深入探讨此类感知误差对框架鲁棒性的影响。
5.  **加权平均的潜在风险**：尽管采用了聚类贝叶斯选择来缓解，但在某些边界情况下，对多个相似但略有差异的动作进行加权平均，理论上仍可能产生一个次优甚至不安全的折中动作。
