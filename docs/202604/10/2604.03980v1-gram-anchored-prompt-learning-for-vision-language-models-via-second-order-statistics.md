## 摘要
参数高效的提示学习已成为将视觉语言模型（VLMs）适配到下游任务的事实标准。现有方法主要侧重于将文本提示与一阶视觉特征（即空间特征图）对齐。虽然这对于细粒度语义区分是有效的，但我们认为仅依赖一阶信息不足以实现稳健的适配，因为这些空间纠缠的特征极易受到领域偏移和局部噪声的影响。在本工作中，我们提出了\textbf{基于二阶统计量的Gram锚定提示学习（GAPL）}用于视觉语言模型，这是一个将局部语义对齐与全局结构一致性协同结合的框架。在方法论上，我们通过\textbf{Gram矩阵}引入了一个额外的二阶统计流，以增强标准的一阶空间交互。通过将提示锚定在这些二阶先验上，我们的方法使语言表示能够动态适应跨不同领域的统计分布偏移。大量实验证明了二阶特征的有效性，并展示了GAPL在各种基准测试上的出色性能。

## Abstract
Parameter-efficient prompt learning has become the de facto standard for adapting Vision-Language Models (VLMs) to downstream tasks. Existing approaches predominantly focus on aligning text prompts with first-order visual features (i.e., spatial feature maps). While effective for fine-grained semantic discrimination, we argue that relying solely on first-order information is insufficient for robust adaptation, as these spatially entangled features are highly susceptible to domain shifts and local noise. In this work, we propose \textbf{Gram-Anchored Prompt Learning (GAPL)} for Vision-Language Models via Second-Order Statistics, a framework that synergizes local semantic alignment with global structural consistency. Methodologically, we introduce an additional second-order statistical stream via \textbf{Gram matrices} that augments the standard first-order spatial interaction. By anchoring prompts to these second-order priors, our approach enables language representations to dynamically adapt to statistical distribution shifts across diverse domains. Extensive experiments indicate the effectiveness of the second-order features, and show compelling performances of GAPL on various benchmarks.

*   **研究背景**：参数高效的提示学习（Prompt Learning）是适配大规模预训练视觉语言模型（如CLIP）到下游任务的主流方法。现有动态提示学习方法（如CoCoOp）主要依赖一阶视觉特征（如全局池化特征或空间特征图）来生成图像条件化的文本提示。
*   **核心问题**：尽管一阶特征在语义区分上有效，但它们对图像的局部外观变化和领域偏移（Domain Shift）非常敏感。当视觉风格改变时，同一语义概念的特征表示可能在特征空间中发生偏移，导致生成的提示不稳定，从而限制了模型的跨域泛化能力。
*   **整体含义**：本文提出，除了依赖一阶语义信息外，引入能够捕捉图像全局结构和风格信息的二阶统计量作为“锚点”，可以增强提示学习的鲁棒性。这旨在解决现有方法在跨域场景下性能下降的问题。

## 2. 方法论：Gram-Anchored Prompt Learning (GAPL)
*   **核心思想**：构建一个三流框架，将传统的基于一阶特征的语义对齐与基于二阶统计量的结构/风格对齐相结合。
    *   **全局不变流 (Global Invariant Stream)**：保持与CLIP预训练模型的稳定语义对齐。
    *   **Gram锚定流 (Gram-Anchored Stream)**：**核心创新**。从视觉编码器的patch tokens计算Gram矩阵 `G = F^T F`，提取其对角线元素 `g_diag = diag(G)` 作为紧凑的图像级二阶统计描述符。通过一个轻量级MLP将其转换为门控向量Γ，用于调制文本特征：`a_style = w + w ⊙ Γ`。该流旨在提供对风格变化不敏感的全局结构锚点。
    *   **上下文锚定流 (Contextual-Anchored Stream)**：利用可学习的局部信号与文本特征进行交互，生成多个上下文文本锚点，实现与局部视觉特征的细粒度对齐。
*   **优化与推理**：
    *   **训练**：采用分支监督和自适应融合策略。每个分支计算分类损失，并引入正则化项防止知识遗忘。
    *   **推理**：根据输入图像自适应的权重融合三个分支的预测结果。

## 6. 主要结论与发现
1.  GAPL框架有效提升了VLM在跨域场景下的鲁棒性。在领域泛化任务中取得了最佳的跨域平均准确率（61.12%），并在基类到新类泛化任务中取得了最佳的平均调和平均数（81.77%）。
2.  引入基于Gram矩阵的二阶统计量作为“风格锚点”是有效的。消融实验表明该模块对提升模型在具有挑战性的域偏移数据集上的性能贡献显著。
3.  三个分支具有互补性：全局流提供稳定基础；Gram流增强跨域鲁棒性；上下文流提升局部判别力。三者结合达到最佳效果。
4.  简单的对角线条目描述符足以捕获有效的二阶信息，使用更复杂的统计量或完整的Gram矩阵并未带来额外收益且计算成本高。
