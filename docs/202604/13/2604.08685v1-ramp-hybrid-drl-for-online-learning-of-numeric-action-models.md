## 摘要
自动规划算法需要一个指定每个动作前提条件和效果的动作模型，但获取这样的模型通常很困难。从观察中学习动作模型是可行的，但现有的数值领域算法是离线的，需要专家轨迹作为输入。我们提出了强化学习、动作模型学习和规划（RAMP）策略，用于通过与环境的交互在线学习数值规划动作模型。RAMP同时训练一个深度强化学习（DRL）策略，从过去的交互中学习一个数值动作模型，并在可能时利用该模型来规划未来的动作。这些组件形成了一个正向反馈循环：RL策略收集数据以改进动作模型，而规划器则生成计划以继续训练RL策略。为了促进RL与数值规划的集成，我们开发了Numeric PDDLGym——一个将数值规划问题转换为Gym环境的自动化框架。在标准IPC数值领域的实验结果表明，在可解性和计划质量方面，RAMP显著优于著名的DRRL算法PPO。

## Abstract
Automated planning algorithms require an action model specifying the preconditions and effects of each action, but obtaining such a model is often hard. Learning action models from observations is feasible, but existing algorithms for numeric domains are offline, requiring expert traces as input. We propose the Reinforcement learning, Action Model learning, and Planning (RAMP) strategy for learning numeric planning action models online via interactions with the environment. RAMP simultaneously trains a Deep Reinforcement Learning (DRL) policy, learns a numeric action model from past interactions, and uses that model to plan future actions when possible. These components form a positive feedback loop: the RL policy gathers data to refine the action model, while the planner generates plans to continue training the RL policy. To facilitate this integration of RL and numeric planning, we developed Numeric PDDLGym, an automated framework for converting numeric planning problems to Gym environments. Experimental results on standard IPC numeric domains show that RAMP significantly outperforms PPO, a well-known DRL algorithm, in terms of solvability and plan quality.

## 1. 核心问题与整体含义
*   **研究背景**：自动化规划（Automated Planning）需要精确的动作模型（Action Model），但手动构建此类模型，尤其是在涉及连续数值变量的**数值规划（Numeric Planning）** 领域，非常困难。
*   **现有局限**：现有的动作模型学习（AML）算法大多为**离线（Offline）** 方法，需要依赖专家提供的执行轨迹作为输入。这限制了其在需要通过与未知环境交互来学习的场景中的应用。
*   **研究动机**：深度强化学习（DRL）虽然擅长在线学习，但在需要长程推理的规划问题上效率不高。因此，本文旨在填补这一空白：**如何在没有先验知识的情况下，通过与环境的在线交互，同时学习一个有效的策略和一个可靠的数值动作模型？**
*   **整体含义**：本文提出了一种名为RAMP的混合策略，首次将DRL、在线AML和符号规划三者结合在一个闭环系统中，用于解决在线数值规划问题。

## 2. 方法论
*   **核心思想 (RAMP Loop)**：
    1.  **DRL探索与数据收集**：使用DRL策略（如PPO）在环境中进行目标导向的探索和交互。
    2.  **动作模型学习 (AML)**：利用收集到的交互轨迹（状态-动作序列），通过离线AML算法（本文选用NSAM/NSAM*）迭代地学习和精炼一个符号化的、安全的数值动作模型。
    3.  **符号规划与执行**：当学到的动作模型足够好时，调用一个经典规划器（如Metric-FF）基于当前状态和目标生成一个计划。如果计划成功生成，则执行该计划；否则回退到DRL策略。
    4.  **正向反馈循环**：规划器生成的“专家”计划为DRL提供了高质量的训练数据；而DRL探索收集的新数据又反过来改进动作模型的准确性和覆盖范围。

*   **关键技术细节**：
    *   **安全模型保证 (Safe Model)**：选用的NSAM算法能保证学到的模型是“安全”的——任何在该模型中有效的计划在真实环境中也是可执行的。这确保了规划的可靠性。
    *   **Numeric PDDLGym框架**：为了解决DRL算法与PDDL2.1描述的符号化数值规划问题之间的接口问题，作者开发了此框架。它能自动将PDDL域和问题转换为标准的Gym环境：
        *   将参数化的符号状态和动作“扁平化”为固定维度的观测向量和离散动作空间。
        *   定义奖励函数和目标终止条件以适配RL训练范式。
    *   **针对PPO的策略更新优化**：
        *   **无效动作屏蔽 (Invalid Action Masking)**：防止智能体重复尝试已知不可行的动作。
        *   **专家行动屏蔽 (Expert Action Masking)**：在执行规划器生成的计划时，将非计划内的行动概率置零。这解决了PPO裁剪机制在面对“专家”行动与当前策略偏好差异过大时梯度更新失效的问题。

##8 .不足与局限
1.**假设限制较强**:依赖于完全可观测、无噪声的环境;要求前提条件和效果均为线性形式;不支持条件效果或数字目标条件等复杂特性.
